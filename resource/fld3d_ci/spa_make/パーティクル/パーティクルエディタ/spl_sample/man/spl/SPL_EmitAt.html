SPL_EmitAt

SPL_EmitAt

構文

#include <spl/spl.h >

void SPL_EmitAt( SPLManager* p_mgr, SPLEmitter* p_emtr, const VecFx32* p_pos ) ;

引数

p_mgr マネージャへのポインタ
p_emtr エミッタへのポインタ
p_pos 生成する位置

返り値

なし

説明

指定したエミッタを指定した位置に移動し、パーティクルを新たに生成します。 指定した位置はリソースに設定されたエミッタオフセット位置に加算されます。 この関数が呼ばれたときは、エミッタのパーティクル生成フラグの如何によらず あらたにパーティクルが生成されます。

この関数はSPL_Calcの前で呼び出すようにしてください。
