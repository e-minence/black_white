完成流れ星ムービー
１ フォトショップで星と星の軌跡用のテクスチャを描きます。
　　 軌跡用のテクスチャはツール上で星が流れていく向きに引き伸ばすので、細かい粒子を描くといいと思います。
２ インテンシティー３bit（８色）アルファ５bit（３２色）に設定。
　　 RGBモードであることを確認。
　　 tga形式（３２bit）で保存。
３　 sptconvフォルダのSPTConv.exeを起動。
　　 フォーマット/a5i3　リピート/リピートなし　フリップ/フリップなしに設定。
　　 tga形式で保存したファイルを、ドラッグ＆ドロップでSPTConv.exeの出力先の下の枠内に移動。
　　 ※星テクスチャは左右対称なので、左半分を描いておき、Sリピート　Sフリップを使うといいと思います。
default.defファイルで設定したテクスチャ読み込み先フォルダに、変換したファイルを移動します。
４　 パーティクルツールを起動します。（起動時にDebug-NITRO-USBダイアログボックス内に作成したテクスチャが読み込まれたか確認）
５ 「エミッタ」ノードを開き、「描画効果」ノードの”テクスチャ”を開きます。
※２Ｄ用エフェクトなので、Ｚ軸を正面に向けて制作すると制作しやすいと思います。
　 親テクスチャメニューから、先ほど作った星テクスチャを選択します。
　 ※Sリピート　Sフリップを使った場合はリピート回数Sを２回にします。
６　 「エミッタタイプ・初速度」の「エミッタ形状」を”円ボリューム”にし、「円タイプの軸」を”Ｙ”にします。
　　 「エミッタサイズ」を”１３０”にします。
エミッタサイズは星が落下してくる範囲になります。
　　 「全方向初速」を”０”にします。
　　 「軸方向初速」を”４.５”にし、「軸X」、「軸Y」を”−０.７”にします。
　　 ※軸XYは流れ星の入射角なので好みの数値でかまいません。
７　 星が画面を通過していくまで表示するので、パーティクルの寿命は長めに設定します。
「生成」の「パーティクル寿命」を”１２０”にし、「パーティクル寿命乱数」を”７０”にします。
　　 「スケール」の「基本スケール」を”６.２”にします。
　　 「スケールアニメ」を”オン”にし、「通常スケール」を”１.２”、「終了スケール」を”０.３”にします。
８　 パーティクルの数が多いので、「放出ステップ」を調整します。
　 「生成」の「放出ステップ」を”７”にしてください。
９　 星の登場と消え方を背景になじませるために、アルファを設定します。
「描画効果」の「アルファ」の「アルファアニメ使用フラグ」を”オン”にします。
「開始アルファ」を”０”にします。
「終了アルファ」を”０”にします。
「アルファイン完了」を”２０”にします。
「アルファアウト開始」を”２３０”にします。
1０ 「チャイルド」ノードの「チャイルド使用フラグ」を”オン”にします。
子テクスチャを星の軌跡用のテクスチャにします。
１１ 流れ星の粒子が広がる様子をチャイルドで表現します。
　 「チャイルド」ノードの「チャイルド設定」の「寿命」を”２０”にします。
「スケール継承率」を”２０”にします。
「スケールアウト使用フラグ」を”オン”にします。
「終了スケール」を”４.５”にします
「アルファアウト使用フラグ」を”オン”にします。
「放出間隔」を”３”にします。
１２　 「チャイルド」ノードの「描画タイプ」を”Ｄビルボード”にします。
「チャイルド設定」の「速度継承率」を”１０”にします。
「生成」の「ＤビルボードＹスケール」を”２.８”にします。
※子パーティクルの「描画タイプ」を「Ｄビルボード」に設定し、親パーティクルの「描画タイプ」を「ビルボード」のままにした場合、「ＤビルボードＹスケール」で子パーティクルだけ引き伸ばすことができます。
１３　 「描画効果」ノードの「カラー」の「カラーアニメ使用フラグ」を”オン”にします。
「開始カラー」、「通常カラー」、「終了カラー」を設定します。
※ランダムカラーをオンにすると「開始カラー」、「通常カラー」、「終了カラー」の３色からランダムに色が選ばれます。
１４　 「エミッタタイプ・初速度」の「初速度乱数」を”５０”にします。
「描画効果」ノードの「スケール」の「スケールイン完了」、「スケールアウト完了」をそれぞれ”１００”にします。
「フィールド」ノードの「重力」の「重力使用フラグ」を”オン”にします。
「強さＹ」を”０.００２”にします。
１５ 「エミッタ」ノードの「エミッタ保存」をクリックします。
「名前をつけて保存」ダイアログのファイル名の欄に名前を入力します。
※ファイルの拡張子は自分で入力します。
　　Go → Home　→ Effect　→ Font　→ Layout　→ Others

レッスン２　流れ星の作成（２Ｄ用）
　
チャイルド パーティクルから放出されるパーティクル。
パーティクルを放出するパーティクルを親パーティクル、放出されたパーティクルを子パーティクルと区別します。
チャイルド機能を使うとパーティクルの数が飛躍的に上がるので、パーティクルの出しすぎに注意！
Ｄビルボード 進行方向を向くビルボード。
速度継承率 親パーティクルの初速を子パーティクルが継承します。
※Ｄビルボードを選択した場合、パーティクルに進行方向を与えないと、パーティクルは表示されません。そのため、ここで子パーティクルに速度継承率を加えます。
スケール継承率 親パーティクルのスケールを子パーティクルが継承します。
スケールアウト使用フラグ 子パーティクルが放出されたと同時に「終了スケール」に向けてスケールアウトするように設定します。
アルファアウト使用フラグ 子パーティクルが放出されたと同時に「アルファ」”０”に向かってアルファアウトするように設定します。
