SPL_LoadTexPlttByVRAMManager

SPL_LoadTexPlttByVRAMManager

構文

#include <spl/spl.h >

BOOL SPL_LoadTexPlttByVRAMManager( SPLManager* p_mgr ) ;

引数

p_mgr マネージャへのポインタ

返り値

成功した際はTRUE、そうでないときはFALSE。

説明

VRAMマネージャを用いてテクスチャパレットをVRAMに読み込みます。 この関数を用いる際には、SPLライブラリはNitroSystemに依存します。 この関数を用いる必要がなく、SPLライブラリとNitroSystemに依存関係を 発生させたくない場合は、SPLをビルドする際にSPL_NOT_USE_NITROSYSTEM=TRUEオプションを 設定してください。この際、この関数は使用できなくなります。

direct形式のテクスチャに関してはなにも処理をおこないません。
