SPL_SetEmitterGlobalColor

SPL_SetEmitterGlobalColor

構文

#include <spl/spl_emitter.h >

void SPL_SetEmitterGlobalColor(SPLEmitter* p_emtr, GXRgb color );

引数

p_emtr エミッタへのポインタ
color 設定色

返り値

なし

説明

指定したエミッタのグローバルカラーを設定します。 最終色は以下のように計算されます
最終色 = ( パーティクルカラー * エミッタカラー ) >> 5
