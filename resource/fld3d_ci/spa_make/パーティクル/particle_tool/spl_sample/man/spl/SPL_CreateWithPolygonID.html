SPL_CreateWithPolygonID

SPL_CreateWithPolygonID

構文

#include <spl/spl.h >

SPLEmitter* SPL_CreateWithPolygonID(SPLManager* p_mgr, int res_no, const VecFx32* p_pos, u16 min_polygonID, u16 max_polygonID );

引数

p_mgr マネージャへのポインタ
res_no リソース番号
p_pos エミッタの位置
min_polygonID 最小ポリゴンID値
max_polygonID 最小ポリゴンID値

返り値

生成されたエミッタへのポインタ。

説明

リソース番号に基づいたリソースをもとにエミッタを生成します。 使用するポリゴンID値の範囲を指定できる点がSPL_Createとは異なります。
