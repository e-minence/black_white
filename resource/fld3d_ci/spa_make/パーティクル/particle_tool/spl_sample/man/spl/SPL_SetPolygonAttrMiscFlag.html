SPL_SetPolygonAttrMiscFlag

SPL_SetPolygonAttrMiscFlag

構文

#include <spl/spl.h >

void SPL_SetPolygonAttrMiscFlag( SPLManager* p_mgr, int flag );

引数

p_mgr マネージャへのポインタ
flag フラグ

返り値

なし

説明

パーティクルのポリゴン属性のmiscフラグを設定します。 指定したマネージャ中で生成されるすべてのパーティクルに対して適用されます。

デフォルトではマネージャのmiscフラグはGX_POLYGON_ATTR_MISC_NONEに設定されています。 フラグに関してはSDKマニュアルのG3_PolygonAttr項をご参照ください。
