SPL_Util_SetCylinderEmiterDirection

SPL_Util_SetCylinderEmiterDirection

構文

#include <spl/spl_emitter.h >

void SPL_Util_SetCylinderEmiterDirection( SPLEmitter* p_emtr, const VecFx32* p1, const VecFx32* p2 );

引数

p_emtr エミッタへのポインタ
p1, p2 始点と終点

返り値

なし

説明

指定した円柱タイプのエミッタの方向と長さを変更します。 p1, p2で指定した2点をつなぐような形状になります。
