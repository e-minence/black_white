はじめに
=======================================================================

TWL-SDK Wi-Fi ライブラリ (以下 TWL-WiFi と呼ぶ) は、TWL-SDK を構成する
パッケージ群の一つとして、インターネットを利用した無線通信の機能を追加
するパッケージです。

TWL-WiFi 単体では動作しませんので、必ず TWL-SDK を事前にインストールして
ください。

また、本体に保存されたネットワーク設定を参照して、無線 AP 接続するための
自動接続機能は TWL-DWC という別のライブラリパッケージに含まれます。
インターネットに接続するアプリケーションは必ず TWL-DWC の接続機能を利用
する必要がありますので、ご注意ください。

TWL-DWC は、ニンテンドーWi-Fiコネクションを利用した通信アプリケーション
を作成するためのライブラリパッケージです。

本パッケージは CodeWarrior for NINTENDO DSi 1.1 対応パッケージです。

=======================================================================
クイックスタート
=======================================================================

1. TWL-SDK などのツールのインストール
   
   所定の手順に従って、SDK や開発機関連のツールをインストールして下さい。
   
   
2. TWL-WiFi パッケージの展開
   
   TWL-SDK とは異なるディレクトリに TWL-WiFi のパッケージを展開します。
   
   
3. 環境変数 TWLWIFI_ROOT の設定
   
   パッケージのトップディレクトリへのパスを環境変数 TWLWIFI_ROOT に設定
   します。以下のようなファイルやディレクトリがあるところがトップディレ
   クトリです。
   
     % ls $TWLWIFI_ROOT
     Makefile  bin  build  docs  include  lib  man  readme.txt  tools
   
   
4. TWL-DWC のインストール
   
   ニンテンドーDSのワイヤレス通信（インフラストラクチャーモード）を利用
   してインターネットに接続し、友達を登録したり、友達の状態を参照したり、
   条件にあった他のニンテンドーDSと接続するためのライブラリである
   TWL-DWC をインストールします。
   インターネットに接続するには、TWL-DWC に含まれる Wi-Fi コネクションへ
   の接続機能を使用する必要があります。
   
   詳細は、TWL-DWC に含まれるドキュメントを参照してください。
