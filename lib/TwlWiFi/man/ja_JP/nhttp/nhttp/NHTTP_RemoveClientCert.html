NHTTP_RemoveClientCert

構文


#include <nitroWiFi/nhttp.h>

int NHTTP_RemoveClientCert( NHTTPConnectionHandle handle );

引数

handle NHTTP_CreateConnection()で生成した接続ハンドル

返り値

クライアント証明書の無効化に成功すれば0を、接続ハンドルがNULLの場合に失敗し-1を返します。

説明

送信すべきHTTPリクエストに対してNHTTP_SetClientCert() またはNHTTP_SetBuiltinClientCert()で設定したクライアント証明書と秘密鍵を無効にします。

参照

NHTTP_SetClientCert, NHTTP_SetBuiltinClientCert
