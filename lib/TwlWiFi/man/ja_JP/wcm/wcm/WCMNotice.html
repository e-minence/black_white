WCMNotice

定義


#include <nitroWiFi/wcm.h>

typedef struct
{
    s16     notify;
    s16     result;
    union
    {
        s32     n;
        void*   p;

    }       parameter[3];

} WCMNotice;

説明

WCMNotify 型の WCM ライブラリからの通知関数に渡される通知の詳細を示す構造体です。 通知種別、処理結果コード、及び詳細パラメータを含みます。

要素

notify 通知の原因となった処理種別が格納されます。詳細については WCM ライブラリ定数を参照して下さい。
result ライブラリ内部で自動的に非同期に行われた処理の結果が格納されます。詳細についてはWCM ライブラリ定数を参照して下さい。
parameter 主にデバッグ用のパラメータが格納されます。WCM ライブラリを使用するにあたって特に意識して頂く必要はありません。
なお、parameter[2] には、通知を発行する原因となった箇所がソースコード( system.c )上の行数として格納されます。
