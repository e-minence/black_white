WCMWepDesc

定義


#include <nitroWiFi/wcm.h>

typedef struct
{
    u8   mode;
    u8   keyId;
    u8   key[80];

} WCMWepDesc;

説明

無線アクセスポイントと DCF 通信する際に使用する WEP 暗号化設定を定義する構造体です。WEP 暗号化種別、データ送信時に使用される WEP キー番号、及び WEP キーデータ列を定義します。

この構造体は NitroSDK WiFi ライブラリとの互換性の為に残されています。通常は、この構造体と互換性を保ちつつ WPA に対応した WCMPrivacyDesc 構造体をご使用下さい。

要素

mode WEP 暗号化種別を指定します。WCM_WEPMODE_NONE を指定すると暗号化しない設定となります。 その他の暗号化設定を指定すると、それぞれの設定に従い key に指定されたデータ列を用いた WEP 暗号化を行う設定となります。 詳細については WCM ライブラリ定数を参照して下さい。
keyId データ送信時に使用する WEP キー番号を 0 〜 3 の範囲で指定します。
key 80 バイトからなる WEP キーデータ列を指定します。20 バイトずつ 4 つの WEP キーデータ列を繋げたデータ列として下さい。 暗号化種別によっては使用されない無駄なデータもありますが、その部分は単に無視されます。
