SOC_IsReleased

構文

#include <nitroWiFi/soc.h>
BOOL SOC_IsReleased ( int s );

引数

s リソースの解放を確認するソケット記述子。

返り値

TRUE ソケットのリソースは解放されています。
FALSE ソケットはクローズ処理中かリソースの解放処理中です。あるいはソケット記述子の値が不正です。 ( s <= 0 )

説明

クローズ処理開始後( SOC_Close が成功した ) のソケットに対してリソースが解放されているか確認します。

SOC_Close の再呼び出しにより同様の確認ができますが、再呼び出し時に他のソケットがリソースを再利用している場合は
SOC_Close は誤ってソケットをクローズします。
