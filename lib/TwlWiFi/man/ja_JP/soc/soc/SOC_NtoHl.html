SOC_HtoNl
SOC_HtoNs
SOC_NtoHl
SOC_NtoHs

構文

#include <nitroWiFi/soc.h>
u32 SOC_HtoNl ( u32 hostlong );
u16 SOC_HtoNs ( u16 hostshort );
u32 SOC_NtoHl ( u32 netlong );
u16 SOC_NtoHs ( u16 netshort );

引数

netlong ネットワークバイトオーダーの 32bit整数。
netshort ネットワークバイトオーダーの 16bit整数。
hostlong ホストバイトオーダーの 32bit整数。
hostshort ホストバイトオーダーの 16bit整数。

返り値

SOCHtoNl()とSOCHtoNs()は、ホストバイトオーダーからネットワークバイトオーダーに変換された引数の値を返します。

SOCNtoHl()とSOCNtoHs()は、ネットワークバイトオーダーからホストバイトオーダーに変換された引数の値を返します。

説明

これらの関数は、16bitまたは32bitの値を、ネットワークバイトオーダーとホストバイトオーダーの間で変換します。
