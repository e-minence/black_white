SOC_GetResolver

構文

#include <nitroWiFi/soc.h>
int SOC_GetResolver( SOCInAddr* dns1, SOCInAddr* dns2 );

引数

dns1 プライマリDNSサーバーアドレス。
dns2 セカンダリDNSサーバーアドレス。

返り値

0 成功。
SOC_ENETRESET ソケットライブラリが初期化されていません。(接続されていません。)

注意：追加のエラーが今後のライブラリリリースで生じたり返されることがあります。すべての負の返り値は汎用のエラーとして扱ってください。

説明

SOC_GetHostByAddr()で使われるプライマリおよびセカンダリのDNSサーバーアドレスを取得します。

参照

SOC_GetHostByAddr, SOCInAddr
