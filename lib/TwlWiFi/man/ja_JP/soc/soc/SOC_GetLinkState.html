SOC_GetLinkState

構文

#include <nitroWiFi/soc.h>
void SOC_GetLinkState ( SOCInterface* interface, BOOL* status );

引数

interface APとリンクしているかどうかを調べるSOCInterfaceへのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。
status インターフェースのリンク状態を受け取るBOOLへのポインタ。 インターフェースが接続状態の場合はTRUEに、接続されていない場合はFALSEが記録されます。

返り値

なし。

説明

指定したネットワーク インターフェースがAPとリンクしているかどうかを調べます。
