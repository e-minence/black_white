SOC_IsNetworkAvailable

構文

#include <nitroWiFi/soc.h>
BOOL SOC_IsNetworkAvailable ( void );

引数

なし。

返り値

ネットワークが利用可能かを真偽値で返します。

説明

ネットワークが利用可能かどうかを調べます。具体的には無線のリンクが確立しているかと、IP アドレスが確定しているかを併せて調べます。

ネットワークが利用可能かどうか調べる方法として SOC_GetHostID 関数の返値を調べる方法もありますが、 SOC_IsNetworkAvailable 関数はノーウェイトで処理を返すという違いがある点に注意して下さい。while ループ内で連続して呼び出すような場合、SOC_GetHostID 関数から単純に置き換えただけでは他のスレッドに処理が回らず正常に動作しない場合があります。
