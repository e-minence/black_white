SOC_GetMtu

構文

#include <nitroWiFi/soc.h>
void SOC_GetMtu ( SOCInterface* interface, s32* mtu );

引数

interface MTUを取得するSOCInterfaceへのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。
mtu インターフェースの最大転送長(MTU)を受け取るs32へのポインタ。

返り値

なし。

説明

指定したインターフェースの最大転送長(MTU)を取得します。

注意：デフォルトの最大転送長は576バイトです。

参照

SOC_GetHostID, SOCConfig
