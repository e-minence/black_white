SOC_InetPtoN

構文

#include <nitroWiFi/soc.h>
int SOC_InetPtoN ( int af, const char* src, void* dst );

引数

af アドレスのファミリー(SOC_PF_INETである必要があります)。
src 渡された文字列へのポインタ。(例 ddd.ddd.ddd.ddd)
dst 数値形式のアドレスを記録するバッファへのポインタ。

返り値

0 成功。
1 src は有効なIPv4アドレス表記ではありません。
SOC_EAFNOSUPPORT afはSOC_PF_INET.ではありません。

説明

IP アドレスをドット10進記法からバイナリアドレスに変換します。

参照

SOC_InetAtoN, SOC_InetNtoA, SOC_InetNtoP
