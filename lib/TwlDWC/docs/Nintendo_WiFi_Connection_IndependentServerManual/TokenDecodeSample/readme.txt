◎動作環境、および必要なPerlモジュール

・Perl v5.8.6にて動作確認を行っています。下記Perlモジュールさえあれば、他のバージョンで
  も動作すると思われます。
・実行には、下記のPerlモジュールが必要です。それぞれ下記のバージョンで動作確認を行ってい
  ますが、他のバージョンでも動作すると思われます。

  - Crypt::CBC          v2.15
  - Crypt::Rijndael_PP  v0.04
  - Digest::MD5         v2.33
  - MIME::Base64        v3.05


◎注意点

・鍵長が128bitであることに注意してください。Crypt::CBCからRijndael_PPを利用する場合、
  Rijndael_PPは鍵長をデフォルト値の256bitとして処理します。これを回避するため、このサン
  プルでは、Rijndael_PP.pmをRijdael_PP_128.pmとしてコピーし、下記の変更を加えたものを使
  用しています。

  ※Rijndael_PP_128.pm 変更点（Rijndael_PP.pm v0.04の場合）
    3c3
    < package Crypt::Rijndael_PP;
    ---
    > package Crypt::Rijndael_PP_128;
    2340c2340
    < sub DEFAULT_KEYSIZE() { 256 }
    ---
    > sub DEFAULT_KEYSIZE() { 128 }
