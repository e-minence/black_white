DWCRnkGetParam_order

定義


#include <dwc.h>

typedef struct DWCRnkGetParam_order {
    u32 sort;
    u32 since;
} DWCRnkGetParam_order;

説明

順位取得モード(DWC_RNK_GET_MODE_ORDER)で取得するときのパラメータを指定します。DWCRnkGetParamのメンバです。

sort DWC_RNK_ORDER_ASC:昇順
DWC_RNK_ORDER_DES:降順
since 何分前からか
