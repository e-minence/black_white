DWCRnkGetParam_friends

定義


#include <dwc.h>

typedef struct DWCRnkGetParam_friends {
    u32 sort;
    u32 limit;
    u32 since;
    s32 friends[DWC_RNK_FRIENDS_MAX];
} DWCRnkGetParam_friends;

説明

友達ランキングリスト取得モード(DWC_RNK_GET_MODE_FRIENDS)で取得するときのパラメータを指定します。DWCRnkGetParamのメンバです。

sort DWC_RNK_ORDER_ASC:昇順
DWC_RNK_ORDER_DES:降順
limit 取得する最大数(2〜65(DWC_RNK_FRIENDS_MAX+1))
since 何分前からか
friends 友達のPIDリスト
