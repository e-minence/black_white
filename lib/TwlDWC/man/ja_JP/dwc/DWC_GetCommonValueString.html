DWC_GetCommonValueString

構文

#include <dwc.h>
int DWC_GetCommonValueString(const char *key,
                             char *value,
                             const char *string,
                             char separator);

引数

key 取得したいvalue文字列に対応するkey文字列へのポインタ
value 取得したvalue文字列の格納先ポインタ
string key/value文字列へのポインタ
separator key/value文字列の、それぞれの文字列の開始を示すための文字

返り値

valueにセットされた文字列長（ヌル文字終端は含まない）。存在しないkeyを指定した場合は-1を返します。

説明

stringで指定されたkey/value文字列から、keyで指定された文字列に対応するvalue文字列を取得します。

key/value文字列とは、separatorを文字列の開始を示す記号として付加し、keyで指定された文字列とvalueで指定された文字列を並べた文字列です。

例えば、string="/key1/value1/key2/value2",key="key1",separator='/'と指定して本関数を呼び出した場合、valueには"value1"という文字列が返されます。

データストレージサーバへのセーブデータを作成する際などに役立ちます。データストレージサーバへのセーブについては、DWC_SavePublicDataAsync/DWC_SavePrivateDataAsync関数を参照してください。

key/value文字列を作成するには、DWC_SetCommonKeyValueString/DWC_AddCommonKeyValueString関数を使います。
