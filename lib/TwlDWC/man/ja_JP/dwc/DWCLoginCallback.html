DWCLoginCallback

構文

#include <dwc.h>
typedef void(*) DWCLoginCallback(DWCError error,
             int profileID,
             void *param);

引数

error DWCエラー種別。内容の詳細は、DWC_GetLastErrorExを参照してください。
profileID 取得できた自分のGSプロファイルID
param DWC_LoginAsync関数で指定したコールバック用パラメータ

返り値

なし

説明

DWC_LoginAsync関数で開始したログイン処理が完了すると呼び出されます。

参照

DWC_LoginAsync

履歴

5.1patch1
用語統一のため「プロファイルID」を「GSプロファイルID」に変更
