DWCRnkData

定義


#include <dwc.h>

typedef struct DWCRnkData {
    u32 order;
    s32 pid;
    s32 score;
    DWCRnkRegion region;
    u32 lastupdate;
    u32 size;
    void * userdata;
} DWCRnkData;

説明

order 順位
pid GameSpyProfileID
score スコア
region リージョンコード
lastupdate 前回の更新からの経過時間(分)
size userdataのサイズ(bytes)
userdata ユーザー定義データのバッファ
