DWCNdFileInfo

定義


#include <dwc.h>

typedef struct DWCNdFileInfo {
    char name[DWC_ND_FILENAME_LEN+1];
    short explain[DWC_ND_FILEEXPLSIN_LEN+1];
    char param1[DWC_ND_FILEATTR_LEN+1];
    char param2[DWC_ND_FILEATTR_LEN+1];
    char param3[DWC_ND_FILEATTR_LEN+1];
    unsigned int size;
} DWCNdFileInfo;

説明

ダウンロードライブラリによって取得されるファイル情報を格納する構造体です。

name ヌル文字終端した最長32文字のASCII文字列で表現されたファイル名
explain ヌル文字終端した最長50文字のUTF16BE文字列で表現されたファイルの説明文
param1 ヌル文字終端した最長10文字のASCII文字列で表現された属性文字列1
param2 ヌル文字終端した最長10文字のASCII文字列で表現された属性文字列2
param3 ヌル文字終端した最長10文字のASCII文字列で表現された属性文字列3
size ファイルサイズ
