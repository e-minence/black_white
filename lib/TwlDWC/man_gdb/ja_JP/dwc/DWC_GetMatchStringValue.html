DWC_GetMatchStringValue

構文

#include <dwc.h>
const char* DWC_GetMatchStringValue(int index,
                                    const char *keyString,
                                    const char *sdefault);

引数

index マッチメイク候補プレイヤーのインデックス。評価コールバックの引数indexをそのまま渡す。
keyString キー識別のための文字列へのポインタ
sdefault 指定したキーを持っていないプレイヤーだった場合のデフォルト文字列
※キーに対応する文字列が空文字の場合やサーバーとの通信中にもこのデフォルト値が返されます。

返り値

マッチメイク候補プレイヤーが持つ、指定したキーに対応する文字列。
キーがない場合は引数sdefaultに渡した文字列が返ってくる。

説明

マッチメイク指標キーに対応する文字列を取得する。

プレイヤー評価コールバック内での使用のみ保証しており、それ以外の場所で呼び出した場合の戻り値は不定。

DWC_AddMatchKeyStringで空文字("")が指定されている場合や、まだサーバーから取得が完了していない状態ではデフォルト値が返されます。

追加した指標キーがサーバに反映されるのに時間がかかるので、アプリケーション側で指標キーをセットしていても、指標キーがないと判断されてデフォルト値が返されることがあります。
