DWCUserRecvCallback

構文

#include <dwc.h>
typedef void(*) DWCUserRecvCallback(u8 aid,
             u8 *buffer,
             int size,
             void *param);

引数

aid 送信元のAID
buffer 受信したデータへのポインタ
size 受信したデータのサイズ
param コールバック用パラメータ

返り値

なし

説明

DWC_SendReliable/DWC_SendReliableBitmap/DWC_SendUnreliable/DWC_SendUnreliableBitmap関数によって送られてきたデータを受信した時に呼び出されます。

本コールバック関数はDWC_SetUserRecvCallback関数によって設定することができます。
