DWCGdbDownloadFileCallback

構文

#include <dwc.h>
typedef void(*) DWCGdbDownloadFileCallback(const void *data,
             int size,
             void *user_param);

引数

data 取得したデータです。
size 取得したデータサイズです。
user_param 非同期処理の実行時に与えたパラメータです。

返り値

なし

説明

DWC_GdbDownloadFileAsyncで引数clear_bufにTRUEを与えていた場合は、引数dataで与えられる領域はコールバックを抜けると解放されてしまいます。FALSEを与えていた場合は解放されませんので、解放してもよいタイミングになったら、DWC_Freeで解放してください。
