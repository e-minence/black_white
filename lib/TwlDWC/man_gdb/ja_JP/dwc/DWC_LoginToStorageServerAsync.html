DWC_LoginToStorageServerAsync

構文

#include <dwc.h>
BOOL DWC_LoginToStorageServerAsync(DWCStorageLoginCallback callback,
                                   void *param);

引数

callback ログイン完了通知コールバック関数へのポインタ
param コールバック用パラメータ

返り値

TRUE ログイン処理開始成功。
FALSE ログイン処理開始失敗。コールバックは呼ばれませんので、注意してください。

説明

データストレージサーバへログインします。DWC_LoadOthersDataAsync,DWC_LoadOwnPrivateDataAsync,DWC_SavePublicDataAsync,DWC_SavePrivateDataAsyncなどのストレージサーバアクセス関数を呼び出す前に、この関数を用いてログインしてください。

本関数呼出し後、DWC_ProcessFriendsMatch関数を呼び続ければログイン処理が進行し、ログイン処理が完了すれば完了通知コールバックが呼び出されます。

データのアクセスが終了した場合、DWC_LogoutFromStorageServer関数を用いて、ログアウトしてください。また、DWC_ShutdownFriendsMatch関数を呼び出すと、自動的にログアウトします。
