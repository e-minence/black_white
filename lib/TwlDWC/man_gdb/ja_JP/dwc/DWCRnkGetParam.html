DWCRnkGetParam

定義


#include <dwc.h>

typedef struct DWCRnkGetParam {
    u32 size;
    union {
        DWCRnkGetParam_order order;
        DWCRnkGetParam_toplist toplist;
        DWCRnkGetParam_nearby nearby;
        DWCRnkGetParam_friends friends;
        u32 data;
    };
} DWCRnkGetParam;

説明

DWC_RnkGetScoreAsyncでスコアを取得するときに指定するパラメータです。DWC_RnkGetScoreAsyncで指定するスコア取得モードに応じて適切なパラメータを設定してください。

size byte単位のサイズ。取得モードに応じたパラメータのサイズを指定します。
例：順位取得モードのときは、sizeof(DWCRnkGetParam_order)を指定します。
order モード=DWC_RNK_GET_MODE_ORDERのときに指定
toplist モード=DWC_RNK_GET_MODE_TOPLISTのときに指定
nearby モード=DWC_RNK_GET_MODE_NEAR_*のときに指定
friends モード=DWC_RNK_GET_MODE_FRIENDSのときに指定
data ライブラリ内で使用
