DWCProxySetting

定義


#include <dwc.h>

typedef struct DWCProxySetting {
    u16 authType;
    u16 port;
    u8 hostName[0x64];
    u8 authId[0x20];
    u8 authPass[0x20];
} DWCProxySetting;

説明

proxy設定情報を格納するための構造体です。

authType proxyの認証タイプ
  • DWC_PROXY_AUTHTYPE_NONE：認証なし
  • DWC_PROXY_AUTHTYPE_BASIC：BASIC認証
port proxyが使用するポート
hostName proxyサーバのURL
authId basic認証時のID
authPass basic認証時のパスワード
