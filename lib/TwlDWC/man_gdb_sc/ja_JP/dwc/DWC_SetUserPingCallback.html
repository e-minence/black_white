DWC_SetUserPingCallback

構文

#include <dwc.h>
BOOL DWC_SetUserPingCallback(DWCUserPingCallback callback,
                             void *param);

引数

callback Pingコールバック関数へのポインタ
param Pingコールバック用パラメータ

返り値

TRUE コールバック関数設定成功
FALSE FriendsMatchライブラリ非動作中で設定失敗

説明

接続されているホストとの往復通信時間（単位：msec）を測定するためのPingコールバックを設定します。DWC_Ping関数を用いて接続中のホストとの往復通信時間を測定開始し、測定結果がPingコールバック関数へ通知されます。
