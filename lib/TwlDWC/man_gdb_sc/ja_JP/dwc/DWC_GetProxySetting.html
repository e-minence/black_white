DWC_GetProxySetting

構文

#include <dwc.h>
BOOL DWC_GetProxySetting(DWCProxySetting *proxy);

引数

proxy 取得したproxy設定情報を書き込む領域へのポインタ。

返り値

TRUE proxy設定の情報の取得に成功しました。
FALSE proxy設定の情報の取得に失敗しました。
失敗の原因は、ユーザー設定４〜６で接続されていない場合、もしくは、proxy設定されていない場合になります。

説明

接続中の接続設定のProxy設定を取得します。
※ただし、proxy設定が可能なユーザー設定４〜６で接続している場合に限られます。

本関数は、自動接続処理で接続状態を監視するDWC_GetInetStatus関数が、DWC_CONNECTINET_STATE_CONNECTEDを返すようになった時点から、DWC_CleanupInet/DWC_CleanupInetAsync関数を呼ぶまでの間で使用できます。
それ以外で呼び出した場合、引数に対してなにも行わずFALSEを返します。
