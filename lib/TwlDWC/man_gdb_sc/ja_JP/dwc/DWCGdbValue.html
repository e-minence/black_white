DWCGdbValue

定義


#include <dwc.h>

typedef union {
    u8 int_u8;
    s16 int_s16;
    s32 int_s32;
    f32 float_f64;
    char * ascii_string;
    u16 * wide_string;
    int boolean;
    DWCTick datetime;
    DWCGdbBinaryData binary_data;
    s64 int_s64;
} DWCGdbValue;

説明

int_u8 1バイトの符号なし整数です。
int_s16 2バイトの符号付き整数です。
int_s32 4バイトの符号付き整数です。
float_f64 8バイトの浮動小数点数です。
ascii_string 一文字1バイトの文字列です。
wide_string 一文字2バイトの文字列です。
boolean 真偽値です。
datetime UnixTimeで表される時間です。
binary_data バイナリデータです。
int_s64 8バイトの符号付き整数です。
