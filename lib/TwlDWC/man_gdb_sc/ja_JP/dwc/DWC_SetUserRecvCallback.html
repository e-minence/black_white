DWC_SetUserRecvCallback

構文

#include <dwc.h>
BOOL DWC_SetUserRecvCallback(DWCUserRecvCallback callback,
                             void *param);

引数

callback 受信完了コールバック関数へのポインタ
param 受信完了コールバック用パラメータ

返り値

TRUE コールバック関数設定成功
FALSE FriendsMatchライブラリ非動作中で設定失敗

説明

データ受信完了コールバックを設定します。本関数で受信完了コールバックを設定しておけば、
DWC_SendReliable/DWC_SendReliableBitmap/DWC_SendUnreliable/DWC_SendUnreliableBitmap関数によって
送られてきたデータを受信した時点で、コールバックが呼び出されます。
