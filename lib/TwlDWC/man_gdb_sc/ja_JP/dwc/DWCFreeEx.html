DWCFreeEx

構文

#include <dwc.h>
typedef void(*) DWCFreeEx(DWCAllocType name,
             void *ptr,
             u32 size);

引数

ptr メモリ確保関数で確保されたバッファのポインタ。
name DWCライブラリ開発用の情報が含まれていますので、無視してください。
size DWCライブラリ開発用の情報で正確な値が格納されていない場合がありますので、無視してください。

返り値

なし

説明

DWCライブラリから使用されるメモリ解放関数です。スレッド排他処理を行う必要があります。

DWCAllocExで確保されたバッファを開放します。NULLポインタが渡されたときには何もせずに処理を終了してください。
