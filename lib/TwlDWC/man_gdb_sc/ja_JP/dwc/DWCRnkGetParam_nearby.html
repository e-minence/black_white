DWCRnkGetParam_nearby

定義


#include <dwc.h>

typedef struct DWCRnkGetParam_nearby {
    u32 sort;
    u32 limit;
    u32 since;
} DWCRnkGetParam_nearby;

説明

周辺ランキングリスト取得モード(DWC_RNK_GET_MODE_NEAR)で取得するときのパラメータを指定します。DWCRnkGetParamのメンバです。

sort DWC_RNK_ORDER_ASC:昇順
DWC_RNK_ORDER_DES:降順
limit 取得する最大数(2〜30(DWC_RNK_GET_MAX))
since 何分前からか
