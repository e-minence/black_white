DWCSvlResult

定義


#include <dwc.h>

typedef struct DWCSvlResult {
    BOOL status;
    char svlhost[DWC_SVL_HOST_LENGTH+1];
    char svltoken[DWC_SVL_TOKEN_LENGTH+1];
} DWCSvlResult;

説明

サービスロケータ情報を格納するための構造体です。

status 対応するサーバの稼動状況です。
TRUE:サーバは稼動しています。
FALSE:サーバは稼動していません。
※この値は将来の拡張のために用意されており、現状は必ずTRUEが格納されます。この値は無視してください。
svlhost 対応するサーバのURLです。
ヌル文字終端した最大64文字のASCII文字列です。
※この値は将来の拡張のために用意されています。この値は無視してください。
svltoken 対応するサーバ向けの認証トークンです。
ヌル文字終端した最大300文字のASCII文字列です。
