WCM_Finish

構文


#include <nitroWiFi/wcm.h>

s32   WCM_Finish( void );

説明

WCM ライブラリを終了させ、初期化前の状態に戻す同期関数です。
WCM_Init()によって初期化した後であり、 かつ WCM ライブラリの内部状態が WCM_PHASE_WAIT である時にのみ実行することができます。

終了処理に成功した場合、WCM_Init()によって再初期化するまでは WCM ライブラリに含まれる API を使用することができなくなりますが、 WCM_Init()で与えたワークバッファは別の用途に使用できるよう解放されます。

引数

ありません。

返り値

以下のいずれかの処理結果を返します。

WCM_RESULT_SUCCESS 終了処理が正常に完了したことを示します。
WCM_RESULT_FAILURE WCM ライブラリが初期化されていない、内部状態が WCM_PHASE_WAIT でない、などの原因で終了処理に失敗したことを示します。

参照

WCM_Init, WCM_CleapupAsync, WCM_TerminateAsync
