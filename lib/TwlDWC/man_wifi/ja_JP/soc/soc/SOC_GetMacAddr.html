SOC_GetMacAddr

構文

#include <nitroWiFi/soc.h>
void SOC_GetMacAddr ( SOCInterface* interface, u8* macAddress );

引数

interface MAC アドレスを取得するSOCInterfaceへのポインタ。 この引数は互換性のため用意されていますが、実際には無視されます。
macAddress MACアドレスを受け取るためのバッファへのポインタ。

返り値

なし。

説明

指定されたインターフェースのMAC (media access control)アドレスを取得します。
