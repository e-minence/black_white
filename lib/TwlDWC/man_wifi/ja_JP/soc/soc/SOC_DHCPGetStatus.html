SOC_DHCPGetStatus

構文

#include <nitroWiFi/soc.h>
#define SOC_DHCP_STATE_REQUESTING    2
#define SOC_DHCP_STATE_BOUND         3    
int SOC_DHCPGetStatus ( SOCDHCPInfo* info );

引数

info ホスト設定を受け取るSOCDHCPInfoへのポインタ(オプション)。この引数は互換性のため用意されていますが、実際には無視されます。

返り値

SOC_DHCP_STATE_BOUND ホストのIPアドレスは設定されています。
SOC_DHCP_STATE_REQUESTING ホストのIPアドレスは設定されていません。

注意：追加のエラーが今後のライブラリリリースで生じたり返されることがあります。すべての負の返り値は汎用のエラーとして扱ってください。

説明

DHCP の状態を取得します。
SOC_DHCPGetStatus()はSOC_Startup()でホストのIPv4アドレスが手動で設定された場合、SOC_DHCP_STATE_BOUNDを返します。
