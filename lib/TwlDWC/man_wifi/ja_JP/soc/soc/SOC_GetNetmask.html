SOC_GetNetmask

構文

#include <nitroWiFi/soc.h>
void SOC_GetNetmask ( SOCInterface* interface, u8* addr );

引数

interface ネットマスクを取得するSOCInterfaceへのポインタ。この引数は互換性のため用意されていますが、実際には無視されます。
addr ソケットにバインドされるアドレスを格納したソケットアドレス構造体へのポインタ。

返り値

なし。

説明

指定されたインターフェースのネットマスクを取得します。
