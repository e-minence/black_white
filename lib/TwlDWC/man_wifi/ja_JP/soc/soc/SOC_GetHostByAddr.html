SOC_GetHostByAddr

構文

#include <nitroWiFi/soc.h>
SOCHostEnt* SOC_GetHostByAddr( const void* addr, int len, int type );

引数

addr ネットワークバイトオーダーでのアドレスへのポインタ。
len addrによってポイントされるアドレスの長さ。
type アドレスのファミリータイプ(IPv4アドレスにはSOC_PF_INET)

返り値

要求されたエントリーが発見されるとSOCHostEnt構造体へのポインタを返します。

説明

アドレスaddrに対応するホストについての情報を返します。

注意：SOC_GetHostByAddr()は再入可能でもスレッドセーフでもありません。

参照

SOC_GetHostByName, SOCConfig, SOCHostEnt
