DWC_WasBuddyFriendData

構文

#include <dwc.h>

BOOL WasBuddyFriendData( const DWCFriendData* frienddata );

説明

引数に指定した友達が、自分との友達関係を破棄したかどうかを調べます。

１度、お互いに友達関係（DWC_IsBuddyFriendData関数でTRUEが返される状態）にあったが、
どちらかが友達関係を破棄したことで、片思いの友達になった場合、友達関係が破棄されたと判定しTRUEを返します。
内部的に、DWC_IsBuddyFriendData関数を使用していますので、 相手の友達情報のフラグは、リアルタイムに更新されているものではないことに注意してください。(本関数は通信を行いません)

引数

frienddata 友達情報の格納されたDWCFriendData構造体のバッファへのポインタ

返り値

TRUE 以前友達関係にあったが、友達は私との友達関係を破棄している。
FALSE そのようなことはない。

参照

DWC_UpdateServersAsync, DWC_IsBuddyFriendData
