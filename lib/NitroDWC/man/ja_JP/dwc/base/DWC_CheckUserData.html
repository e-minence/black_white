DWC_CheckUserData

構文

#include <dwc.h>

BOOL DWC_CheckUserData( const DWCUserData* userdata );

説明

ユーザデータの正当性を、ユーザデータに含まれるCRC値を用いてチェックします。セーブされたユーザデータをメモリに読み込んだ際の確認に使用してください。

ユーザデータをセーブする前には必ずDWC_ClearDirtyFlag関数でDirtyフラグをクリアし、ユーザデータに含まれるCRC値を再計算しておいてください。また、セーブされたユーザデータをメモリに読み込んだ後、本関数を呼び出す前にDWC_ClearDirtyFlag関数を呼び出すと本来の意味を成さないことに注意してください。

引数

userdata ユーザデータが格納されたDWCUserData構造体へのポインタ

返り値

TRUE 正当
FALSE 正当ではない

参照

DWC_CreateUserData, DWC_ClearDirtyFlag
