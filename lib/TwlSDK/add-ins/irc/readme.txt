●IRC ライブラリについて

IRC(赤外線カード) を使用するためのライブラリです。


●各ファイルについて

<<デモ build/demos/irc/connect_irc>>

IRC 同士の通信機能と IRC バックアップデバイスへのアクセスを示すサンプルです。


●ライブラリの使用法

まず IRC ライブラリパッケージ全体を任意のディレクトリに展開し、
展開先の irc ディレクトリへのパスを環境変数 TWL_ADDIN_IRC_ROOT に指定します。
(指定しない場合は $(TWLSDK_ROOT)/add-ins/irc として扱われます)

その後 $(TWL_ADDIN_IRC_ROOT)/build/buildtools へ移動して make を 1 回だけ
実行することにより、これ以降 make で作成するすべてのプロジェクトは
自動的に IRC ライブラリのインクルードパスとライブラリリンクを
参照できるようになります。


CodeWarrior IDE でビルドする場合などは以下のインクルードパスとライブラリを
プロジェクトに追加指定する必要があります。

インクルードパス：$(TWL_ADDIN_IRC_ROOT)/include
リンクライブラリ：$(TWL_ADDIN_IRC_ROOT)/lib/$(TWL_LIBTYPE)/libirc$(TWL_LIBSUFFIX).a
