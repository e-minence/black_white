IRC_Send

構文

#include <irc.h>

void IRC_Send(u8 *buf, u8 size, u8 command);

引数

*buf 送信したいデータの先頭アドレス。コマンドのみ送りたい場合はNULLを指定してください。
size 送信するデータのバイト数。
command 通信内容を識別するためのコマンド番号。偶数のみ使用可能ですが、内容はアプリケーションで自由に設定してください。

返り値

なし

説明

データを送信するための関数です。一度に128バイトまで送信可能です。相手にデータが届いた際にIRC_SetRecvCallbackで指定したコールバックが相手側で呼ばれます。

参照

IRC_SetRecvCallback
