赤外線通信カードライブラリ(IRC) API 機能一覧

ライブラリ機能

IRC_Init ライブラリの初期化を行います。
IRC_Move 通信処理を進めるプロセス関数です。
IRC_Send データを送信する関数です。
IRC_SetRecvCallback データ受信時のコールバック関数を設定する関数です。
IRC_Connect 接続要求をする関数です。
IRC_SetConnectCallback リンク接続時のコールバック関数を設定する関数です。
IRC_IsConnect 通信リンクが張れているかどうか確認する関数です。
IRC_IsSender 通信開始を行った端末かどうか調べる関数です。
IRC_Shutdown ライブラリの終了処理を行います。
IRC_SetShutdownCallback リンク切断時のコールバック関数を設定する関数です。

変数型・構造体

IRCRecvCallback データ受信時に呼ばれるコールバック関数の形式です。
IRCConnectCallback リンク接続時に呼ばれるコールバック関数の形式です。
IRCShutdownCallback リンク切断時に呼ばれるコールバック関数の形式です。
