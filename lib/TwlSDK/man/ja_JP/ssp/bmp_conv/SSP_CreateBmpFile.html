SSP_CreateBmpFile

構文


u32 SSP_CreateBmpFile(const void* src, void* dst, u16 width, u16 height, SSPConvertBmpType type);

引数

src 入力データ(YUV422 もしくは RGB555 形式)
dst 作成した BMP ファイルを格納するバッファ
width 画像の幅
2 の倍数である必要があります
height 画像の高さ
type 入力データの種類(SSP_BMP_YUV422, SSP_BMP_RGB555)

返り値

作成された BMP ファイルのサイズを返す。

説明

YUV422 形式、もしくは RGB555 形式の画像データから、BMP ファイルを作成します。
作成される BMP ファイルの種類は Windows bitmap になります。
画像の横サイズは 2 の倍数である必要があります。
