SSP_ExtractJpegDecoderExif

構文


inline BOOL SSP_ExtractJpegDecoderExif(u8* data, u32 size);

引数

data Exif情報を抽出するJPEGファイル
size Exif情報を抽出するJPEGファイルのサイズ

返り値

成功ならば、TRUE を返す

説明

JPEGファイルから Exif 情報を抽出します。
デコード処理は行わないため SSP_StartJpegDecoder() を呼び出した場合よりも早く取得できます。

実際に Exif に含まれていた情報を取得するには、本関数を呼び出した後に、各々の Get 関数を使用してください。
