SSP_StartJpegDecoder

構文


BOOL SSP_StartJpegDecoder(u8* data, u32 size, void* dst, s16* width, s16* height, u32 option);

引数

data デコードするJPEGファイル
size デコードするJPEGファイルのサイズ
dst デコードした画像を格納するバッファ
width * height * 2 のサイズのバッファが必要となります。
width デコードを許す画像の最大横幅(pixel)を指定する。
デコード完了後は、デコードした画像の横幅が格納されている。
height デコードを許す画像の最大縦幅(pixel)を指定する。 デコード完了後は、デコードした画像の縦幅が格納されている。
option デコードオプション (SSP_JPEG_THUMBNAIL)

返り値

成功ならば、TRUE を返す

説明

JPEGファイルを RGB555 形式の画像データにデコードします。
デコードできるJPEGファイルは YUV420、YUV422、YUV444 のものに限られます。

option に SSP_JPEG_THUMBNAIL を指定した場合はサムネイル画像のデコードを行い、何も指定しなかった場合( 0 を指定した場合)は、メイン画像のデコードを行います。
