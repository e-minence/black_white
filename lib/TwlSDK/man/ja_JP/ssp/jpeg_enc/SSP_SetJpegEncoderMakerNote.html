SSP_SetJpegEncoderMakerNote

構文


static inline void SSP_SetJpegEncoderMakerNote(u8 *buffer, u16 size);

引数

buffer エンコード時に画像に埋め込む、DSiカメラ用のデータです。
size エンコード時に画像に埋め込む、DSiカメラ用のデータサイズです。

返り値

なし。

説明

JPEGのメーカーノート部分に埋め込むDSiカメラ用のデータを登録します。
本関数は SSP_SetJpegEncoderMakerNoteEx()を SSP_MAKERNOTE_PHOTO タグで 呼び出した場合と同じ動作になります。
ユーザー独自のデータを埋め込みたい場合には、SSP_SetJpegEncoderMakerNoteEx() を SSP_MAKERNOTE_USER で使用するようにして下さい。

注意：
SSP_StartJpegEncoder() の完了後に、エンコードの成功、失敗問わず本関数で設定されたデータはクリアされます。
複数の画像をエンコードする場合は、それぞれのエンコードの前にデータを登録する必要があることに注意してください。

参照

SSP_StartJpegEncoder, SSP_SetJpegEncoderMakerNoteEx
