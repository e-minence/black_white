SSP_GetJpegEncoderBufferSize

構文


u32 SSP_GetJpegEncoderBufferSize(u32 width, u32 height, u32 sampling, u32 option);

引数

width エンコードする画像の横幅(pixel)
height エンコードする画像の縦幅(pixel)
sampling 主画像の出力形式 (SSP_JPEG_OUTPUT_YUV444 / SSP_JPEG_OUTPUT_YUV420 / SSP_JPEG_OUTPUT_YUV422)
option エンコードオプション (SSP_JPEG_RGB555 / SSP_JPEG_YUV422 / SSP_JPEG_THUMBNAIL)

返り値

エンコードに必要なバッファサイズを返す

説明

SSP_StartJpegEncoder() でエンコードを行う際に、作業用バッファとして必要となるサイズを取得します。
本関数に与える引数については、SSP_StartJpegEncoder() に与えるものと同じものとするようにしてください。

参照

SSP_StartJpegEncoder
