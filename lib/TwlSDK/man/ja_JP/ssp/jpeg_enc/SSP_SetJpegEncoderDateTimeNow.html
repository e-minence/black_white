SSP_SetJpegEncoderDateTimeNow

構文


BOOL SSP_SetJpegEncoderDateTimeNow(void);

引数

なし。

返り値

成功した場合は TRUE を返します。
失敗した場合は FALSE を返します。

説明

JPEGに埋め込む撮影日時情報に現在の日時を登録します。
本関数は内部で RTC_GetDateTime() を呼び出した後に、SSP_SetJpegEncoderDateTime() と同等の処理を行っているため、事前に RTC_Init() を呼び出す必要があります。
本関数が 0 以外を返す場面は RTC_GetDateTime() が失敗した場合のみとなります。

この情報は、IFD0 の DateTime と、ExifIFD の DateTimeOriginal タグ、DateTimeDigitized タグとして登録されます

注意：
SSP_StartJpegEncoder() の完了後に、エンコードの成功、失敗問わず本関数で設定された日時情報はクリアされます。
複数の画像をエンコードする場合は、それぞれのエンコードの前に日時情報を登録する必要があることに注意してください。
