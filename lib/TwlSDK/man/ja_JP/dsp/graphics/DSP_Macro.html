DSP Graphics コンポーネントのマクロ

定義


#define DSP_CALC_SCALING_SIZE(value, ratio) ((u32)(value * (u32)(ratio * 1000) / 1000))
#define DSP_CALC_SCALING_SIZE_FX(value, ratio) ((u32)( value * (u32)(ratio * 1000 / 4096.0f) / 1000))

説明

DSP_CALC_SCALING_SIZE(value, ratio) DSP_Scaling*() を用いて、縦または横のサイズ valueの画像を ratio(f32型)倍拡縮した場合の結果を計算する。
DSP_CALC_SCALING_SIZE_FX(value, ratio) DSP_ScalingFx*() を用いて、縦または横のサイズ valueの画像を ratio(fx32型)倍拡縮した場合の結果を計算する。

参照

DSP_Scaling*, DSP_ScalingFx*
