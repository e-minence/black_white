DSP_EncodeG711

構文


void DSP_EncodeG711(void *dst, const void *src, u32 len, DSPAudioCodecMode mode);

引数

dst A-law8bitまたはu-law8bitの変換データを格納するバッファ。
指定するバッファには16ビットDMA転送と同様の制限があります。詳細は後述の「注意」をご参照ください。
src PCM16bitの変換元データを保持しているバッファ。
指定するバッファには16ビットDMA転送と同様の制限があります。詳細は後述の「注意」をご参照ください。
len 変換すべきサンプル数。
mode 変換すべき符号化方式。
DSP_AUDIO_CODEC_MODE_G711_ALAWまたはDSP_AUDIO_CODEC_MODE_G711_ULAWのいずれかが選択可能です。

返り値

なし。

説明

PCMサンプルをG.711符号化方式で符号化します。
処理の完了を確認するには DSP_WaitForG711() か DSP_TryWaitForG711() を使用することが出来ます。

注意

  • この関数を使用する前に DSP_LoadG711() を呼び出して G.711コンポーネントをロードしておく必要があります。
  • 転送元・転送先に指定するバッファはいずれも16ビットDMA転送と同等の条件を満たしている必要があります。
    • アドレスは16bitの整数倍である必要があります。
    • DSPに処理させる変換元のバッファは、事前にキャッシュをストアしておく必要があります。
    • DSPから得られた変換後のバッファは、参照する前にキャッシュを無効化する必要があります。
