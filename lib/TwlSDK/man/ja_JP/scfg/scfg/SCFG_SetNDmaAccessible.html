SCFG_SetNDmaAccessible　

構文

#include <twl/scfg.h>
static inline void SCFG_SetNDmaAccessible( BOOL sw );

引数

sw 新規DMA ブロックへのアクセス制御です。
TRUE ならばアクセス可能となります。
FALSE ならばアクセス不可能となります。

返り値

なし。

説明

新規DMAブロックへAHB からアクセス可能かどうかを設定します。

sw をFALSE にすると新規DMAブロックへのアクセスが不可能となります。初期値は 「アクセス可能」です。アクセス不可能にしても、バスから見えなくなるだけで、それ以前に設定した値自体は有効です。

安全のため、フラグの値の変更は 新規DMAブロックへのアクセスが発生していないときに行ってください。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
