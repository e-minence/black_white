SCFG_IsConfigBlockAccessible　

構文

#include <twl/scfg.h>
static inline BOOL SCFG_IsConfigBlockAccessible( void );

引数

なし。

返り値

TRUE ならば アクセス可能。
FALSE ならば アクセス不可能。

説明

ARM9 のコンフィグブロックへのアクセスが可能かどうかを取得します。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。

SCFG概要
