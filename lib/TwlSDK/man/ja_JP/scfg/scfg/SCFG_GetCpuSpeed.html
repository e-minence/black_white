SCFG_GetCpuSpeed

構文

#include <twl/scfg.h>
static inline SCFGCpuSpeed SCFG_GetCpuSpeed( void );

引数

なし。

返り値

設定されている ARM9 のCPU動作速度。
SCFG_CPU_SPEED_1X ならば NITRO と同じ 67.03MHz
SCFG_CPU_SPEED_2X ならば NITRO の倍速の 134.06MHz

説明

現在の、ARM9-CPU の周波数設定を取得します。

SCFG ブロックは TWL ハードウェア上にしかありませんが、この関数は NITRO ハードウェア上で動作させても正しく値を返します (SCFG_CPU_SPEED_1X が返ります)。
