SCFG_IsClockSuppliedToCamera　

構文

#include <twl/scfg.h>
void SCFG_IsClockSuppliedToCamera( void );

引数

なし。

返り値

TRUE ならば、カメラインターフェースブロックへのクロック供給がされています。
FALSE ならば、クロックは供給されていません。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。

説明

カメラインターフェースブロックへのクロック供給の設定を取得します。
