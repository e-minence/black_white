SCFG_SetIntcExpanded　

構文

#include <twl/scfg.h>
static inline void SCFG_SetIntcExpanded( BOOL sw );

引数

sw ARM9 の INTC回路をNITRO-CPU から拡張するかどうかです。
TRUE ならば拡張します。
FALSE ならば以前のNITRO用 INTC 回路と同じになります。

返り値

なし。

説明

ARM9の割り込みコントローラ(INTC)回路を拡張するかどうかを設定します。

TWLモードでは、初期値は 「拡張する」 となります。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
