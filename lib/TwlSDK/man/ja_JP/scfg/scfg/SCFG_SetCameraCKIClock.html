SCFG_SetCameraCKIClock　

構文

#include <twl/scfg.h>
static inline void SCFG_SetCameraCKIClock( BOOL sw );

引数

sw CAM_CKI クロックの制御フラグ。
TRUE ならば CAM_CKI イネーブル (16.67MHz を出力)
FALSE ならば CAM_CKI ディセーブル (Low固定出力)

返り値

なし。

説明

カメラ用 CAM_CKI クロックを設定します。

引数 sw が TRUE ならば CAM_CKI イネーブルとなり、16.76MHz のクロックを出力します。FALSE ならば CAM_CKI ディセーブルとなり、Low 固定出力となります。

安全のため、カメラモジュールの動作中に CAM_CKI ディセーブルとしないでください。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
