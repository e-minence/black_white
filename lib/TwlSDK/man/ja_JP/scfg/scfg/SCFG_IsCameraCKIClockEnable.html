SCFG_IsCameraCKIClockEnable　

構文

#include <twl/scfg.h>
static inline BOOL SCFG_IsCameraCKIClockEnable( void );

引数

なし。

返り値

CAM_CKI クロックの制御フラグ。
TRUE ならば CAM_CKI イネーブル (16.67MHz を出力)
FALSE ならば CAM_CKI ディセーブル (Low固定出力)

説明

現在の、カメラ用 CAM_CKI クロックの設定を取得します。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
