SCFG_IsCameraAccessible　

構文

#include <twl/scfg.h>
static inline BOOL SCFG_IsCameraAccessible( void );

引数

なし。

返り値

TRUE の場合、カメラインターフェースブロックへアクセスが可能です。
FALSE の場合、カメラインターフェースブロックへのアクセスは不可能です。

説明

カメラインターフェースブロックへAHB からアクセス可能かどうかの設定を取得します。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
