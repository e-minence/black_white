SCFG_IsRendererFixed　

構文

#include <twl/scfg.h>
static inline BOOL SCFG_IsRendererFixed( void );

引数

なし。

返り値

現在の設定を返します。
TRUE ならば回路の修正を行います。
FALSE ならば回路の修正を行いませんのでNITRO-CPU と同じ動作となります。

説明

レンダラー回路を修正する設定となっているかを取得します。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
