SCFG_IsIntcExpanded　

構文

#include <twl/scfg.h>
static inline BOOL SCFG_IsIntcExpanded( void );

引数

なし。

返り値

ARM9 の INTC回路を NITRO-CPU から拡張する設定となっているかどうか。
TRUE ならば 拡張します。
FALSE ならば 以前の NITRO用INTC回路を同じになります。

説明

ARM9 の割り込みコントローラ(INTC)回路をNITRO-CPUから拡張する設定になっているかどうかを取得します。

(注意)
この関数は TWL ハードウェア上で動作しているときに使用してください。

NITRO ハードウェアには SCFG ブロックが存在していませんので、その上で呼び出されることは想定していません。
