PhotoChecker

概要

PhotoChecker は、TWL 本体内の photo 領域をチェックする TWL アプリケーションです。
TCL ライブラリ経由で書き込まれていない不正なファイルが存在していないかどうかをチェックすることが可能です。

実行

以下のいずれかの方法で実行することができます。

  • IS-TWL-DEBUGGER ソフトウェアを使用して PhotoChecker.srl を実行する。
  • PhotoChecker.srl を TWL カードに書き込んで実行する。

実行を開始すると、photo: 以下に存在する画像が上画面に、ファイル名が下画面に次々と表示されていきます。
photo: 以下が不正な状態であった場合、上画面が赤色になります。
正常な状態であった場合は、上画面が緑色になります。

以下のような場合、不正な状態とみなされます。

  • DSi カメラを一度も起動していない等の理由で管理ファイルが存在しないにも関わらず、その他のファイルが存在した場合
  • 管理ファイルの情報が不正な場合
  • DSi カメラ、及び TCL を経由せずに書き込んだファイルが存在した場合
  • DSi カメラ、及び TCL を経由して書き込んだファイルを DSi カメラを経由せずに削除した場合

具体的には以下のエラーメッセージが表示されます。

There is no file which suited the management file. 管理ファイルに登録されているファイルが見つからなかった場合。
There is no management file and there are other files. 管理ファイルが存在しないにも関わらず、その他のファイル、ディレクトリが見つかった場合。
There is a file which is not in the management file. 管理ファイルに登録されていないファイルが見つかった場合。
There is a file of an illegal pathname. DSiカメラの命名規則に則っていない名前のファイルが見つかった場合。
There is a directory of an illegal pathname. DSiカメラの命名規則に則っていない名前のディレクトリが見つかった場合。
This management file is not right. 管理ファイルの内容が正常でなかった場合。
This picture is illegal size. デコードに成功した画像のサイズがDSiカメラが定めるサイズ(640x480)ではなかった場合。
The signature is not added to this JPEG file. 電子署名が付加されていなかった場合、もしくは電子署名は付加されているがデコードできない画像だった場合。
Memory allocation error. 通常発生しないエラー。
Can not open JPEG file. 通常発生しないエラー。
Can not read JPEG file. 通常発生しないエラー。
Can not read management file. 通常発生しないエラー。
Can not open directory. 通常発生しないエラー。
The return of entry information went wrong. 通常発生しないエラー。
Does not reach so far. 通常発生しないエラー。
