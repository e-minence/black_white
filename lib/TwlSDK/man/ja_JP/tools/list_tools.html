ツール機能一覧

ROM イメージ関連

makelcf リンカコマンドファイルをテンプレートファイルより生成します。
makelcf.TWL
compstatic アプリケーションの常駐領域及びオーバーレイを圧縮します。
compstatic.TWL
makerom ROM スペックファイルより ROM イメージを作成します。
makerom.TWL
セーブデータサイズ一覧 NAND アプリでセーブデータサイズに指定可能な値一覧です。
maketad SRL ファイルから tad を作成します。
cmptad 2 つの tad ファイルを比較します。
tad2rom tad ファイルを内包する SRL ファイルを生成します。
makebanner ROM バナーファイルを作成します。
makebanner.TWL
bannercvtr NITRO-CHARACTER や ntexconv で作成したファイルからmakebanner.TWL 用のファイルに変換します。
NITRO-CHARACTER によるバナー作成手順 NITRO-CHARACTER を使用してバナーのアニメーションデータを作る手順を解説しています。
addbanner ROM バナーファイルをテスト表示するための ROM ファイルを生成します。
showversion ROM ファイルにリンクされている TWL-SDK のバージョンを表示します。
attachsign ダウンロードプレイにおける送信ファイル（子機イメージ）に認証コードを貼り付けます。
buryarg プログラム中から利用できる引数データをROMイメージに書き込みます。
buryarg.TWL
loadrun エミュレータにROMバイナリを読み込んで実行します。
loadrun.TWL
nitrorun buryarg と loadrun を一度に呼び出すスクリプトファイルです。
nitrorun.TWL buryarg.TWL と loadrun.TWL を一度に呼び出すスクリプトファイルです。

データ変換関連

ntrcomp ファイルを TWL-SDK で扱える形式で圧縮します。
compBLZ ファイルを TWL-SDK で扱える後方展開可能な BLZ 形式で圧縮します。
ntexconv BMP/TGA 形式の画像ファイルを TWL-SDK で扱える形式に変換します。
ppmconv* PPM 形式の画像ファイルを C ソース形式などに変換します。
bin2obj バイナリファイルをリンカでリンク可能なオブジェクト形式に変換します。
xml2env XML 形式で書かれた ENV ライブラリ用リソースデータを、C のソースファイル形式に変換します。
init2env INIT 形式で書かれた ENV ライブラリ用リソースデータを、C のソースファイル形式に変換します。
packnlf
packtlf		 nlf / tlf ファイルが参照している全てのファイル指定を相対パス化し、それら全てをまとめ ZIP ファイルとして出力します。
rc4conv CRYPTO ライブラリで扱える RC4 アルゴリズムによる暗号化を行います。
aesconv AES 暗号アルゴリズムに基づく暗号化/復号化を行います。
makelst ELライブラリを使用する際に使用する、静的モジュールのシンボル情報を C のソースファイル形式で出力します。
stripdebug ELライブラリを使用してリンクする動的モジュールをオブジェクトファイルから生成します。

デモ用ツール関連

mic2wav マイクサンプルデモmic-2が生成するテキストデータを .wav ファイルに変換します。
pdic2c.pl 図形認識デモcharacterRecognition-2が出力する図形パターンデータを C 言語のソースファイルに変換します。
