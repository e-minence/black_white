PM_SetExitCallbackInfo

構文

#include <twl/spi.h>
void PM_SetExitCallbackInfo(
　　　　 PMExitCallbackInfo* info,
　　　　 PMExitCallback callback,
　　　　 void* arg );


typedef void (*PMGenCallback) (void *);
#define PMExitCallback PMGenCallback

引数

info セットするコールバック情報構造体へのポインタ
callback コールバック関数
arg コールバックの引数

返り値

なし。

説明

終了コールバック情報を設定します。

コールバック関数 callback と、その引数 arg を、 info で示したコールバック情報構造体 に登録します。この後 PM_AppendPreExitCallback() などでこの info を指定して終了コールバック登録を行ってください。

内部では、info のメンバに callback と arg をセットするだけの処理を行っています。

参照

PM_AppendPreExitCallback
