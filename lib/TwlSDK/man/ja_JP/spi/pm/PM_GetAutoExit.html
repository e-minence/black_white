PM_GetAutoExit

構文

#include <twl/spi.h>
BOOL PM_GetAutoExit( void );

引数

なし。

返り値

TRUE ならば自動的に終了する設定となっています。
FALSE ならばユーザが PM_ReadyToExit() を呼ぶまで終了が保留されます。

説明

終了処理後に自動的にハードウェアリセットまたは電源オフを行うかどうかの設定値を取得します。

PM_SetAutoExit() でユーザがどのように設定しているかを取得します。詳細は PM_SetAutoExit() を参照してください。

参照

PM_SetAutoExit, PM_ReadyToExit
