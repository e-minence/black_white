MBErrorStatus

定義

#include <nitro/mb.h> 

typedef struct
{
    u16 errcode;
} MBErrorStatus;

説明

MB_CommSetParentStateCallback() のコールバックにMB_COMM_PSTATE_ERROR が返されたときにポインタで渡される、
エラー要因の格納された構造体です。
各メンバの意味は以下の通りです。
メンバ 意味
errcode MBライブラリのエラーコードを示すMBErrCode型の値です。
