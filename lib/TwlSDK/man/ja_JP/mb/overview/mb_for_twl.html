TwlSDK MB ライブラリ (overview)

MB ライブラリは、TWL や NTR 本体を問わず NTR プログラムを配信することができるライブラリです。

マルチブート配信プログラム

マルチブート親機から子機へ配信できるプログラムは、NITRO ROM、HYBRID ROM です。
TWL 本体のみでしか動作しない LIMITED ROM は配信ができません。

従って、MB ライブラリを用いたダウンロードプレイでは、カメラや DSP などの TWL 固有の機能を使用することはできません。

また、クローンブートを使用する場合、上記のように LIMITED ROM は配信できないため、親機の ROM は NITRO ROM もしくは HYBRID ROM の必要があります。
親機側でカメラなど TWL 固有機能を使用したい場合は HYBRID ROM を選択して下さい。

注意

MB ライブラリが配信するのは、NITRO ROM の部分のみです。
HYBRID ROM が持っている TWL 専用 実行バイナリといった拡張領域は配信されず、子機側で受信するプログラムは NITRO ROM 相当のものとなります。
