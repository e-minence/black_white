WXC_Start

構文

#include <nitro/wxc.h>

void WXC_Start(void);

引数

なし

返り値

なし

説明

WXCライブラリ内部のワイヤレス駆動を開始します。
２回以上呼び出された場合は何も行いません。
この関数の呼び出しの中で、ライブラリの内部状態はWXC_STATE_READYから WXC_STATE_ACTIVEに遷移します。

ワイヤレスを起動する前でしか設定できない機能については、 WXC_Init関数の呼び出しからこの関数の呼び出しまでの間に関数を呼び出します。

参照

WXCStateCode
WXC_Init WXC_End WXC_Stop
