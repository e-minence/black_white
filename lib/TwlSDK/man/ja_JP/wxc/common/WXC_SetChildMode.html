WXC_SetChildMode

構文

#include <nitro/wxc.h>

void WXC_SetChildMode(BOOL enable);

引数

enable 子機側でしか起動させない場合は TRUE を指定します。
通常の設定に戻す場合は FALSE を指定します。

返り値

なし

説明

ワイヤレスを子機側としてしか起動させないよう設定します。
この関数はWXC_Start関数より前に呼び出す必要があります。
通常の用途においてはこの関数を使用する必要はありません。

参照

WXC_Init WXC_Start WXC_End
