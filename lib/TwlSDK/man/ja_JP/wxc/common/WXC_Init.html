WXC_Init

構文

#include <nitro/wxc.h>

void WXC_Init(void *work, WXCCallback callback, u32 dma);

引数

work WXCの内部状態管理のために割り当てるメモリへのポインタ
このサイズはWXC_WORK_SIZEバイト以上である必要があります。
割り当てたメモリはWXC_End関数を呼ぶまで内部で使用されます。
callback WXCライブラリからの通知を受けるコールバック関数
dma WXCライブラリ内部で使用するDMAチャンネル

返り値

なし

説明

WXCライブラリを初期化します。
全てのWXC関数の呼び出しの前に少なくとも１回呼び出す必要があります。
２回以上呼び出された場合は何も行いません。
この関数の呼び出しの中で、ライブラリの内部状態はWXC_STATE_ENDから WXC_STATE_READYに遷移します。

この関数の呼び出しの後、実際に内部のワイヤレスを起動するためにWXC_Start関数を呼び出す必要があります。

参照

WXCCallback WXCStateCode WXC_WORK_SIZE
WXC_Start WXC_End
