WXC_SetInitialData

構文

#include <nitro/wxc.h>

void WXC_SetInitialData(u32 ggid, u8 *send_ptr, int send_size, u8 *recv_ptr, int recv_size)

引数

ggid データブロックを関連付けるGGID
send_ptr 送信するデータが格納されたバッファ
send_size send_ptrのデータサイズ
送信可能なデータサイズは最大WXC_MAX_DATA_SIZEバイトです。
recv_ptr データを受信するバッファ
recv_size recv_ptrのデータサイズ
送信可能なデータサイズは最大WXC_MAX_DATA_SIZEバイトです。

返り値

なし

説明

WXC_RegisterCommonData関数で指定したデータブロックの内容を再設定します。
各引数の内容はWXC_RegisterCommonData関数と同様です。

WXCCallback
WXC_RegisterCommonData WXC_AddData WXC_UnregisterData
