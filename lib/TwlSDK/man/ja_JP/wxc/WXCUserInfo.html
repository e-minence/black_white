WXCUserInfo

定義

#include <nitro/wxc.h>

/* 各AIDごとにWXCが保持しているユーザ情報 (WXC_GetUserInfo() で取得) */
typedef struct WXCUserInfo
{
    u16     aid;
    u8      macAddress[6];
}
WXCUserInfo;


要素

aid 現在接続中のユーザのAIDです。
macAddress 現在接続中のユーザのMACアドレスです。

説明

WXCライブラリが各AIDごとに保持しているユーザ情報です。

参照

WXC_GetUserInfo
