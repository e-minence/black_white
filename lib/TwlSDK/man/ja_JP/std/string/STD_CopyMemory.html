STD_CopyMemory

構文

#include <nitro/std.h>

void* STD_CopyMemory(void *destp, const void *srcp, u32 size);
void* STD_MemCpy(void *destp, const void *srcp, u32 size);

引数

src 転送元アドレス
dest 転送先アドレス
size 転送サイズ

返り値

なし。

説明

CPUでメモリコピーを行います。C標準関数の memcpy() と同じ動作をします。

STD_MemCpy() は STD_CopyMemory() の別名として define されています。

VRAMにアクセスする際には、MI_CpuCopy8(), MI_CpuCopy16(), MI_CpuCopy32(), MI_CpuCopyFast() の何れかを使って下さい。

参照

MI_CpuCopy*, STD_MoveMemory
