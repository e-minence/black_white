STD_SearchCharReverse

構文

#include <nitro/std.h>

char* STD_SearchCharReverse( char* srcp, int c );

char* STD_StrRChr( char* srcp, int c );

引数

srcp 探される文字列のポインタを指定します。
c 探す文字を指定します。

返り値

文字列 srcp の中で文字 c が最後に現れる位置へのポインタを返します。
それがない場合には NULL を返します。

説明

文字列 srcp から、文字 c が最後に現れる場所を探します。 C標準関数の strrchr() と同じ動作をします。

STD_StrRChr() は STD_SearchCharReverse() の別名として define されています。
