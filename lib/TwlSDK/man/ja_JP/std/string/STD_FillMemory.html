STD_FillMemory

構文

#include <nitro/std.h>

void* STD_FillMemory(void *destp, u8 data, u32 size);
void* STD_MemSet(void *destp, u8 data, u32 size);

引数

destp 転送先アドレス
data 転送データ
size 転送サイズ

返り値

なし。

説明

CPUでメモリを指定のデータで埋めます。

STD_MemSet() は STD_FillMemory() の別名として define されています。

VRAMにアクセスする際には、MI_CpuFill8(), MI_CpuFill16(), MI_CpuFill32(), MI_CpuFillFast() の何れかを使って下さい。

参照

MI_CpuFill*
