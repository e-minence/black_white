STD_SearchChar

構文

#include <nitro/std.h>

char* STD_SearchChar( char* srcp, int c );

char* STD_StrChr( char* srcp, int c );

引数

srcp 探される文字列のポインタを指定します。
c 探す文字を指定します。

返り値

文字列 srcp の中で文字 c が最初に現れる位置へのポインタを返します。
それがない場合には NULL を返します。

説明

文字列 srcp から、文字 c を探します。 C標準関数の strchr() と同じ動作をします。

STD_StrChr() は STD_SearchChar() の別名として define されています。
