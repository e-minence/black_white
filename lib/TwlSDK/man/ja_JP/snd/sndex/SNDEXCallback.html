SNDEXCallback

定義

#include <twl/snd.h>
typedef void ( *SNDEXCallback ) ( SNDEXResult result, void* arg );

引数

result 非同期処理の結果を示すSNDEXResult列挙型の値です。 通知されうる値はコールバックの要因となる処理毎に異なりますので、各非同期関数の説明を参照して下さい。
arg 非同期関数を呼び出す際に指定された引数です。

返り値

なし。

説明

サウンド拡張機能を非同期的に操作する関数によって起動された非同期処理が完了した際に呼び出されるユーザコールバック関数の型式です。 result引数にはSNDEXResult列挙型の処理結果が、arg引数には非同期関数を呼び出す際に指定した引数が、それぞれ渡されます。

参照

SNDEX_GetMuteAsync, SNDEX_GetI2SFrequencyAsync, SNDEX_GetDSPMixRateAsync, SNDEX_GetVolumeAsync, SNDEX_SetMuteAsync, SNDEX_SetI2SFrequencyAsync, SNDEX_SetDSPMixRateAsync, SNDEX_SetVolumeAsync
