SNDEXFrequency

定義

#include <twl/snd.h>
typedef enum
{
    SNDEX_FREQUENCY_32730 = 0,
    SNDEX_FREQUENCY_47610 = 1

} SNDEXFrequency;

要素

SNDEX_FREQUENCY_32730 32.73kHz
SNDEX_FREQUENCY_47610 47.61kHz

説明

I2S動作周波数を示す列挙型です。
