SNDEXHeadphone

定義

#include <twl/snd.h>
typedef enum
{
    SNDEX_HEADPHONE_UNCONNECT   =   0,
    SNDEX_HEADPHONE_CONNECT     =   1

} SNDEXHeadphone;

要素

SNDEX_HEADPHONE_UNCONNECT ヘッドホン非接続状態。
SNDEX_HEADPHONE_CONNECT ヘッドホン接続状態。。

説明

ヘッドホンの接続状態を示す列挙型です。

参照

SNDEX_IsConnectedHeadphoneAsync, SNDEX_IsConnectedHeadphone
