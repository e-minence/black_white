NA_InitSharedFont　

構文

#include <twl/na.h>
BOOL NA_InitSharedFont( void );

引数

なし。

返り値

正常に処理が完了した場合は TRUE が返ります。
関数が失敗する原因としては以下が考えられます。

  • アプリケーションが NAND アプリではない。
  • ファイルハンドルが全て使用されており、空きがない。
  • ニンテンドー DS 上で動作している。

説明

本体内蔵フォントを使用するための準備処理を行います。
他の本体内蔵フォント関連 API より前に一度だけ呼び出す必要があります。

この関数は NAND アプリ専用です。カードアプリでは使用できません。
