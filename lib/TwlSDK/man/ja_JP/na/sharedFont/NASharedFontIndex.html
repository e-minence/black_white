NASharedFontIndex　

定義

#include <twl/na.h>
typedef enum NASharedFontIndex
{
    NA_SHARED_FONT_WW_L = 0,
    NA_SHARED_FONT_WW_M = 1,
    NA_SHARED_FONT_WW_S = 2,
    NA_SHARED_FONT_MAX  = 3

    NA_SHARED_FONT_CN_L = NA_SHARED_FONT_MAX,
    NA_SHARED_FONT_CN_M = 4,
    NA_SHARED_FONT_CN_S = 5,
    NA_SHARED_FONT_KR_L = 6,
    NA_SHARED_FONT_KR_M = 7,
    NA_SHARED_FONT_KR_S = 8,
    NA_SHARED_FONT_CN_KR_MAX = 9
}
NASharedFontIndex;

要素

NA_SHARED_FONT_WW_L 日米欧豪向け本体内蔵フォント大を表すインデックスです。
NA_SHARED_FONT_WW_M 日米欧豪向け本体内蔵フォント中を表すインデックスです。
NA_SHARED_FONT_WW_S 日米欧豪向け本体内蔵フォント小を表すインデックスです。
NA_SHARED_FONT_CN_L 中国向け本体内蔵フォント大を表すインデックスです。
NA_SHARED_FONT_CN_M 中国向け本体内蔵フォント中を表すインデックスです。
NA_SHARED_FONT_CN_S 中国向け本体内蔵フォント小を表すインデックスです。
NA_SHARED_FONT_KR_L 韓国向け本体内蔵フォント大を表すインデックスです。
NA_SHARED_FONT_KR_M 韓国向け本体内蔵フォント中を表すインデックスです。
NA_SHARED_FONT_KR_S 韓国向け本体内蔵フォント小を表すインデックスです。

説明

内蔵フォント関連 API で内蔵フォントを指定するために使用されるインデックスです。
