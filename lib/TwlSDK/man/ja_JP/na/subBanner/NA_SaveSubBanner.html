NA_SaveSubBanner　

構文

#include <twl/na.h>
BOOL NA_SaveSubBanner( const NASubBanner *banner );

引数

banner 設定するサブバナーが格納されたバッファへのポインタ

返り値

正常に処理が完了した場合は TRUE が返ります。
関数が失敗する原因としては以下が考えられます。

  • banner の crc16_anime メンバが不正である。
  • banner の anime メンバ が不正である。
  • アプリケーションが NAND アプリではない。
  • アプリケーションがサブバナーの仕様を宣言していない。
  • ファイルハンドルが全て使用されており、空きがない。
  • ニンテンドー DS 上で動作している。

説明

サブバナーを指定されたサブバナーデータに置き換え、 サブバナーががランチャーで表示されるようにします。

この関数は NAND アプリ専用です。カードアプリでは使用できません。

banner の anime メンバ が不正である場合、DEBUG版では警告が出力されます。
出力された内容とNASubBannerを参照して修正してください。
