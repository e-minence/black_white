CARD ライブラリ(CARD)サンプルデモ

場所

$TwlSDK/build/demos/card

コンパイル方法

$TwlSDK/build/demos/cardディレクトリに移り、

% make

と入力します。コンパイルが始まり、実行ファイルである .srl ファイルやバイナリ情報を記述した .nef または .tef ファイルが、

$TwlSDK/build/demos/card/(サンプル名)/bin/(プラットフォーム)/(ターゲット)/

の下に生成されます。例えば、

$TwlSDK/build/demos/card/backup/bin/ARM9-TS/Release/main.srl

が backup デモの実行ファイルとなります。

実行方法

NITRO ROM の場合

Ensata エミュレータを用いる場合

  • ensata で *.nef, *.srl を実行する。

IS-NITRO-DEBUGGER を用いる場合

  • IS-NITRO-DEBUGGER で *.nlf ファイルを読みこんで実行する。
  • IS-NITRO-DEBUGGER で *.srl を実行する。
  • コマンドラインから loadrun.exe で IS-NITRO-DEBUGGER で *.srl を実行する。

IS-TWL-DEBUGGER を用いる場合

  • IS-TWL-DEBUGGER で *.nlf ファイルを読みこんで実行する。
  • IS-TWL-DEBUGGER で *.srl を実行する。
  • コマンドラインから loadrun.TWL.exe で IS-TWL-DEBUGGER で *.srl を実行する。

TWL/NITRO HYBRID ROM の場合

IS-NITRO-DEBUGGER を用いる場合

  • IS-NITRO-DEBUGGER で *.srl を実行する。
  • コマンドラインから loadrun.exe で IS-NITRO-DEBUGGER で *.srl を実行する。

IS-TWL-DEBUGGER を用いる場合

  • IS-TWL-DEBUGGER で *.tlf ファイルを読みこんで実行する。
  • IS-TWL-DEBUGGER で *.srl を実行する。
  • コマンドラインから loadrun.TWL.exe で IS-TWL-DEBUGGER で *.srl を実行する。

TWL LIMITED ROM の場合

IS-TWL-DEBUGGER を用いる場合

  • IS-TWL-DEBUGGER で *.tlf ファイルを読みこんで実行する。
  • IS-TWL-DEBUGGER で *.srl を実行する。
  • コマンドラインから loadrun.TWL.exe で IS-TWL-DEBUGGER で *.srl を実行する。

説明

デモ名 ディレクトリ 動作
backup demos CARD ライブラリによるバックアップデバイスへのアクセス方法を示す、ごく簡単な雛形です。
card-1trom demos ROM種類をワンタイムPROMに指定して転送速度をデバッグ出力するサンプルです。
card-mrom demos ROM種類をマスクROMに指定して転送速度をデバッグ出力するサンプルです。
(カートリッジデモに含まれる)
ctrdg/pullout-1		 demos このサンプルはカートリッジデモ ($TwlSDK/build/demos/ctrdg) に含まれます。
カードとカートリッジの抜け検出を行います。

動作

card-mrom は NTR モードでしか正常動作しません。
