RTC_Init

構文

#include <nitro/rtc.h>
void RTC_Init( void );

引数

なし。

返り値

なし。

説明

リアルタイムクロックAPI の為の初期化を行います。
他のリアルタイムクロック操作関数を呼び出す前に一回呼び出して下さい。

注意

リアルタイムクロックはバッテリーによってデータが保持されており、本体の電源が入っていない時も継続して時間を計測しています。 初期化関数はこの保持されているデータを初期化しませんので、日付・時刻の初期値は一定ではないことに注意して下さい。 なお、アラームに関する設定値は本体の起動シーケンス内で全て 0 クリアされますが、 ソフトウェアリセットによるアプリケーションの再起動時には再起動前の設定が持ち越されます。

リアルタイムクロックの日付・時刻はエンドユーザが本体設定機能を用いて自由に調整できます。 このため、本体が起動し直された時には、再起動の前と後とで日付・時刻の一貫性は保証されないことに注意して下さい。

RTC_SetAlarmStatus, RTC_SetAlarmParam
