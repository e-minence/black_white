RTCTimeEx

定義

#include <nitro/rtc.h>
typedef struct {
         u32        hour;
         u32        minute;
         u32        second;
         u32        millisecond;
} RTCTimeEx;

要素

hour 時を示す 0〜23 の数値
minute 分を示す 0〜59 の数値
second 秒を示す 0〜59 の数値
second ミリ秒を示す 0〜999 の数値

説明

ミリ秒精度の時刻を表せるよう RTCTime を拡張した構造体です。
時分秒ミリ秒を示します。

参照

RTCTime, RTC_GetDateTimeExFromSWClock
