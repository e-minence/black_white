RTC_GetSWClockTick

構文

#include <nitro/rtc.h>
OSTick RTC_GetSWClockTick( void );

引数

なし。

返り値

ソフトウェアクロックから計算される 2000年1月1日午前0時0分0秒から現時点までの総チック値を返します。
最後に行ったソフトウェアクロックの同期に失敗している場合は 0 を返します。

説明

ソフトウェアクロックから計算される 2000年1月1日午前0時0分0秒から現時点までの総チック値を OSTick 型（u64）で返します。

最後に呼び出した RTC_SyncSWClock() が失敗している場合は 0 を返します。

参照

RTC_SyncSWClock
