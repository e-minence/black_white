AESCallback　

構文

#include <twl/aes.h>

typedef void (*AESCallback)(AESResult result, void* arg);

要素

result AES 処理の実行結果。
arg ユーザ定義のパラメータ。コールバック関数を引数とする関数において、コールバック関数の次の引数が渡されます。

説明

AES の処理が完了したときに呼び出されるコールバック関数の型です。

result には以下のいずれかの値が渡されます。

  • AES_RESULT_SUCCESS
  • AES_RESULT_VERIFICATION_FAILED
  • AES_RESULT_BUSY
