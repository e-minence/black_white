AES 暗号 高位 API (overview)

説明

概要

AES 暗号 高位 API は基本 API の処理のうち暗号化に必要なカウンタ初期値及び Nonce を ライブラリ内部で適切に生成するようにした API です。 出力されるデータは鍵を除く復号化に必要なデータをひとまとめに格納していますので、 暗号化関数の出力を復号化関数に渡すだけで復号化できるようになっています。

乱数生成器

AES 暗号 高位 API ではライブラリ内部でのカウンタ初期値または Nonce の生成のため、 AES を使用した乱数生成器を実装しています。
この乱数生成器を単独で使用することもできますが、その強度は保障されません。
