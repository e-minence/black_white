G2_SetBG2Affine

構文

#include <nitro/gx/g2.h>

void G2_SetBG2Affine(
    const MtxFx22 * mtx,
    int centerX,
    int centerY,
    int x1,
    int y1
);

引数

mtx 変換行列へのポインタ
centerX 回転中心座標のX成分
centerY 回転中心座標のY成分
x1 回転前座標のX成分（Hオフセットの設定）
y1 回転前座標のY成分（Vオフセットの設定）

返り値

なし。

説明

BG2のアフィン変換を設定します。

スクリーン上の点 (centerX, centerY) を中心とする回転・拡大/縮小は以下の式で求められます。

平行移動は、スクリーンに対し、表示画面とのオフセット (x1, y1) をドット単位で指定します。

(例)
BG2スクリーンを点 (centerX, centerY) を中心に変換行列 mtx で回転・拡大/縮小し、
x 軸方向に dx，y 軸方向に dy 平行移動します。


G2_SetBG2Affine(
    mtx,
    centerX, centerY,
    dx, dy );
