SVC_CompareSHA1

構文

#include <twl/os/common/systemCall.h>
int SVC_CompareSHA1( const void* decedHash,
                       const void* digest );

引数

decedHash SHA-1 値１
digest SHA-1 値2

返り値

SHA-1 値の比較が一致した場合、TRUEを返します。
一致しない場合、または引数に NULL が与えられた場合、FALSE を返します。

説明

２つのSHA-1 の結果を比較します。

内部では、decedHash と digest で示された領域から、20バイトの比較を行います。
