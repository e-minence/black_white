EL_Export

構文

#include <twl/el.h>

BOOL EL_Export( ELAdrEntry* AdrEnt );

引数

AdrEnt エクスポート用のアドレステーブル

返り値

シンボル情報の追加に成功したならば TRUE を返す。

説明

アドレステーブルにシンボル情報を追加します。
静的モジュール側のシンボルを動的モジュール側から呼び出せるようにするために、静的モジュールのシンボル情報を ELライブラリに渡すために使用される関数です。
通常は、makelstツールで生成されるEL_AddStaticSym() の内部で 呼ばれるようになっているため、上記関数内以外で本関数を呼び出す必要はありません。

参照

EL_AddStaticSym, ELAdrEntry
