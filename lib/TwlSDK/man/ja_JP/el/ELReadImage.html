ELReadImage

定義

#include <twl/el.h>


typedef u32   (*ELReadImage)( u32 offset, void* buf, u32 size);

要素

offset ライブラリに渡すべき動的モジュールデータの先頭からのオフセット
buf データをロードする先のアドレス
size ロードするサイズ

説明

ELライブラリが動的モジュールを読み込む際に使用する関数です。
この関数では、リンクする動的モジュールの先頭アドレス + offset の位置から size 分だけのデータを buf にロードする必要があります。
読み出しに成功した場合は 0を返すようにしてください。
読み出しに失敗した場合には 0以外を返すようにしてください。
