ELLinkMode

定義

#include <twl/el.h>

typedef enum ELLinkMode {
  EL_LINKMODE_ROUGH = 0,
  EL_LINKMODE_ONESHOT,
  EL_LINKMODE_RELATION
} ELLinkMode;

要素

EL_LINKMODE_ROUGH 現在は使用できません。
EL_LINKMODE_ONESHOT 動的オブジェクトが外部参照している場合、全てベニヤを前提として計算するモードです。
EL_LINKMODE_RELATION 現在は使用できません。

説明

動的リンクをテストする際のモードを表す列挙型です。
EL_CalcEnoughBufferSizeforLink*()で指定することができます。
