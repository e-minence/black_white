FSDateTime

定義


typedef struct FSDateTime
{
    u32     year;    // 0-
    u32     month;   // 1-12
    u32     day;     // 1-31
    u32     hour;    // 0-23
    u32     minute;  // 0-59
    u32     second;  // 0-60
}
FSDateTime;

説明

FSライブラリで扱う時刻情報をあらわす構造体です。
各メンバの内容は以下の通りです。

year年をあらわす0以上の値です。
month月をあらわす1から12の値です。
day日をあらわす1から31の値です。
hour時をあらわす0から23の値です。
minute分をあらわす0から59の値です。
second秒をあらわす0から60の値です。
