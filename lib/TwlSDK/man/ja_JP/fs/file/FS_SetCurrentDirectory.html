FS_SetCurrentDirectory

構文


#include <nitro/fs.h>

BOOL FS_SetCurrentDirectory(const char *path);

引数

path カレントディレクトリに指定するパス

返り値

カレントディレクトリを正しく設定できればTRUEを、失敗すればFALSEを返します。
失敗時の詳細なエラーはFS_GetResultCode()で取得することができます。

説明

カレントディレクトリを設定します。
FSライブラリの関数呼び出しに相対パスを指定した場合、この関数で指定されたカレントディレクトリを参照します。

この関数は、指定されたディレクトリが現時点で存在していなくともアーカイブが正しく存在していればTRUEを返します。

注意

  • この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
    また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
