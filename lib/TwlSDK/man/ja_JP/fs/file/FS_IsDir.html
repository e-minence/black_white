FS_IsDir

構文

#include <nitro/fs.h>

static inline BOOL FS_IsDir(volatile const FSFile *p_file);

引数

p_file FSFile構造体のアドレス

返り値

ディレクトリ位置情報を格納しているなら TRUE を、そうでないなら FALSE を返します。

説明

指定したファイルが現在ディレクトリ位置情報を格納しているか判定します。
ディレクトリ位置情報は、FS_FindDir() または FS_SeekDir() で格納され、
FS_OpenFileEx() 、FS_OpenFileFast() 、および FS_CloseFile() で破棄されます。
