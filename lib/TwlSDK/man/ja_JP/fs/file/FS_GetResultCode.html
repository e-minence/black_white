FS_GetResultCode

構文

#include <nitro/fs.h>

static inline FSResult FS_GetResultCode(volatile const FSFile *p_file);

引数

p_file FSFile構造体のアドレス

返り値

ファイルが最後に行った処理の結果を返します。

説明

指定したファイルが最後に行った処理の結果を返します。
まだ処理を行っていないファイルや処理中のファイルに対してこの関数を呼び出した場合、結果は未定義です。

特定のアーカイブが最後に行った処理の結果を取得したい場合は、FS_GetArchiveResultCode()をご使用ください。
