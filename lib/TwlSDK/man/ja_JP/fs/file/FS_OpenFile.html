FS_OpenFile

構文

#include <nitro/fs.h>

BOOL FS_OpenFile(FSFile *file, const char *path);

引数

file FSFile構造体のアドレス
path ファイルのパス名

返り値

ファイルが正しく開ければ TRUE を, そうでないなら FALSE を返します。

説明

この関数は、FS_OpenFileEx(file, path, FS_FILEMODE_R) と等価です。
互換性のために用意されていますが、現在ではFS_OpenFileEx関数の使用を推奨します。

注意

開いたファイルの構造体はシステムリソースに関連付けられるため、 別の構造体にそのままコピーしても情報の移動や複製はできません。 詳細はFSFile構造体をご参照ください。
