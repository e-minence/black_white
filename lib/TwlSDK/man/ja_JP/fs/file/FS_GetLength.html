FS_GetLength

構文


#include <nitro/fs.h>

u32	FS_GetLength(FSFile *file);

引数

file ファイルハンドルを保持したFSFile構造体

返り値

ファイルのサイズを返します。

説明

開いているファイルのサイズを取得します。
この関数はFS_GetFileLength関数と等価です。
互換性のために用意されていますが、現在ではFS_GetFileLength関数の使用を推奨します。
