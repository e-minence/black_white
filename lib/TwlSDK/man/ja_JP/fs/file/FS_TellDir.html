FS_TellDir

構文

#include <nitro/fs.h>

BOOL FS_TellDir( const FSFile *p_dir, FSDirPos *p_pos );

引数

p_dir FSFile構造体のアドレス
p_pos ディレクトリ位置を格納する構造体のアドレス

返り値

現在位置を示す FSDirPos の値を格納して常に TRUE を返します。

説明

指定したディレクトリリストが現在保持している列挙位置を、FSDirPos として取得します。
この値を使用して FS_SeekDir() を呼び出すことにより、
再び現在の列挙位置を再現することができます。

参照

FSFile FSDirPos FS_FindDir FS_ReadDir FS_SeekDir FS_RewindDir
