FS_GetArchiveResource

構文


#include <nitro/fs.h>

BOOL FS_GetArchiveResource(const char *path, FSArchiveResource *resource);

引数

path アーカイブを特定できるパス名
resource 取得したリソース情報を格納するFSArchiveResource構造体

返り値

リソース情報が正常に取得できればTRUEを、失敗すればFALSEを返します。
失敗時の詳細なエラーはFS_GetArchiveResultCode()で取得することができます。

説明

指定したパスが属するアーカイブの現在のリソース情報を取得します。

この関数は主に、FATフォーマットのアーカイブに対して空き容量を確認するために利用します。NANDアプリのセーブデータサイズを決定する際には、別途「セーブデータサイズ一覧」の情報もご参考ください。

注意

  • この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
    また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
