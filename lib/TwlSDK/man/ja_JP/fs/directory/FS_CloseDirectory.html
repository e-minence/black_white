FS_CloseDirectory

構文


#include <nitro/fs.h>

BOOL FS_CloseDirectory(FSFile *dir);

引数

dir FS_OpenDirectory()で開いたディレクトリのハンドル

返り値

ディレクトリを正しく閉じればTRUEを、失敗すればFALSEを返します。
失敗時の詳細なエラーはFS_GetResultCode()で取得することができます。

説明

開いているディレクトリを閉じます。

注意

  • この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
    また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
