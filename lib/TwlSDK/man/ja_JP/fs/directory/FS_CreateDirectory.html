FS_CreateDirectory

構文


#include <nitro/fs.h>

BOOL FS_CreateDirectory(const char *path, u32 permit);

引数

path 生成するディレクトリのパス名
permit ディレクトリへのアクセス権
FS_PERMIT_*のフラグを組み合わせて指定します。
(将来のシステム拡張のための予約であり、現状ではアクセス権は無視されます)

返り値

ディレクトリが正常に生成されればTRUEを、失敗すればFALSEを返します。
失敗時の詳細なエラーはFS_GetArchiveResultCode()で取得することができます。

説明

指定したディレクトリを新規に生成します。
以下の場合には失敗してFALSEを返します。

  • 対象となるアーカイブがディレクトリの新規生成に対応できない場合
  • すでに同名のディレクトリが存在する場合

注意

  • この関数を割り込みハンドラ(IRQモード)から呼び出すことはできません。
    また、割り込みを禁止したままの状態にあると処理が完了しない場合がある点に注意して下さい。
