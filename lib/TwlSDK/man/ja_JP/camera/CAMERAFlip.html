CAMERAFlip

定義

#include <twl/camera.h>


typedef enum
{
    CAMERA_FLIP_NONE,
    CAMERA_FLIP_VERTICAL,
    CAMERA_FLIP_HORIZONTAL,
    CAMERA_FLIP_REVERSE,

    CAMERA_FLIP_MAX
}
CAMERAFlip;

要素

CAMERA_FLIP_NONE 反転処理を行いません。
CAMERA_FLIP_VERTICAL 上下反転を行います。
CAMERA_FLIP_HORIZONTAL 左右反転を行います。
CAMERA_FLIP_REVERSE 180度回転を行います。

説明

カメラの出力データに対する反転処理を指定するために使用する列挙型です。
