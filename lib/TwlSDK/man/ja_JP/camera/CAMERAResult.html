CAMERAResult

定義

#include <twl/camera.h>


typedef enum CAMERAResult
{
    CAMERA_RESULT_SUCCESS = 0,
    CAMERA_RESULT_SUCCESS_TRUE = 0,
    CAMERA_RESULT_SUCCESS_FALSE,
    CAMERA_RESULT_BUSY,
    CAMERA_RESULT_ILLEGAL_PARAMETER,
    CAMERA_RESULT_SEND_ERROR,
    CAMERA_RESULT_INVALID_COMMAND,
    CAMERA_RESULT_ILLEGAL_STATUS,
    CAMERA_RESULT_FATAL_ERROR,
    CAMERA_RESULT_MAX
}
CAMERAResult;

要素

CAMERA_RESULT_SUCCESS 処理成功であり、値の意味としてvoidを表す。
CAMERA_RESULT_SUCCESS_TRUE 処理成功であり、値の意味としてTRUEを表す。
CAMERA_RESULT_SUCCESS_FALSE 処理成功であり、値の意味としてFALSEを表す。
CAMERA_RESULT_BUSY 他のリクエストを実行中。
CAMERA_RESULT_ILLEGAL_PARAMETER 不正なパラメータ。
CAMERA_RESULT_SEND_ERROR カメラコマンドの送信に失敗。
CAMERA_RESULT_INVALID_COMMAND 不正なコマンド。
CAMERA_RESULT_ILLEGAL_STATUS カメラの再起動処理が実行中であることを表す。
CAMERA_RESULT_FATAL_ERROR その他何らかの原因で処理に失敗。

説明

カメラ処理の結果を表す列挙型です。
