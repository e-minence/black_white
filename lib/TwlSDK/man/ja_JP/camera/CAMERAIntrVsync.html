CAMERAIntrVsync

定義

#include <twl/camera.h>


typedef enum {
    CAMERA_INTR_VSYNC_NONE          = (0 << REG_CAM_CNT_IREQVS_SHIFT),
    CAMERA_INTR_VSYNC_NEGATIVE_EDGE = (2 << REG_CAM_CNT_IREQVS_SHIFT),
    CAMERA_INTR_VSYNC_POSITIVE_EDGE = (3 << REG_CAM_CNT_IREQVS_SHIFT)
} CAMERAIntrVsync;

要素

CAMERA_INTR_VSYNC_NONE 割り込み無効。
CAMERA_INTR_VSYNC_NEGATIVE_EDGE カメラVSYNCの立ち下がりで割り込み発生。
CAMERA_INTR_VSYNC_POSITIVE_EDGE カメラVSYNCの立ち上がりで割り込み発生。

説明

カメラのVSYNC割り込みのタイミングを指定するために使用する列挙型です。
