MI_WriteByte

構文

#include <nitro/mi.h>

void MI_WriteByte( void* address, u8 data )

引数

address バイトデータを書き込むアドレス(ポインタ)
data 書き込むデータ

返り値

なし。

説明

void型ポインタで指定されたアドレス address の先に、バイトデータdata を書き込みます。

この関数は過去のバージョンとの互換性のために用意されています。現在では単に address を u8ポインタとみなして data を書き込むという inline 関数でしかありません。
