MI_ReadByte

構文

#include <nitro/mi.h>

u8 MI_ReadByte( const void* address )

引数

address バイトデータを読むアドレス(ポインタ)

返り値

指定アドレスのメモリ内容(u8)

説明

const void 型ポインタで指定されたアドレス address の先のメモリ内容をu8データで取得します。

この関数は過去のバージョンとの互換性のために用意されています。現在では単に address を u8ポインタとみなして data を読むという inline 関数でしかありません。
