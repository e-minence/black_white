MI_SetNDmaConfig

構文

#include <twl/mi.h>

void MI_SetNDmaConfig( u32 ndmaNo, MINDmaConfig *config );

引数

ndmaNo DMA 番号です。0〜3の値を取ります。
config コピー元のMINDmaConfig 構造体へのポインタです。

返り値

なし。

説明

MIライブラリ内で保持している、TWL新規DMAの DMA config 構造体にユーザ指定の構造体からコピーします。

ndmaNo で指定した番号の DMAに関する設定に、config で示した領域からコピーします。

内部動作

内部の DMA config 領域を MI_CpuCopy32() を用いてコピーしています。

参照

MI_InitNDmaConfig, MI_GetNDmaConfig
