MI_GetWramBankEnable*

構文

#include <twl/mi.h>

MIWramEnable MI_GetWramBankEnable( MIWramPos wram, int num )
MIWramEnable MI_GetWramBankEnable_A( int num )
MIWramEnable MI_GetWramBankEnable_B( int num )
MIWramEnable MI_GetWramBankEnable_C( int num )

引数

wram イネーブル／ディセーブル情報を取得する対象の WRAM
num イネーブル／ディセーブル情報を取得する対象 WRAM のスロット番号

返り値

指定のWRAM領域のイネーブル／ディセーブル情報です。
MI_WRAM_ENABLE、MI_WRAM_DISABLE のいずれかです。

説明

指定のWRAM領域のイネーブル／ディセーブル情報を取得します。

wram はイネーブル／ディセーブル情報を取得する対象となる WRAM で、MIWramPos 型です。

num は対象の WRAM のスロット番号で、WRAM-A は 0〜3、WRAM-B と WRAM-C は 0〜7 となります。

内部動作

IOレジスタのMBK1〜MBK5 にアクセスしています。

概要(ワークRAM)、MI関連の定数
