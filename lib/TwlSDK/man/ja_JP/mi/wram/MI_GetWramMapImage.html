MI_GetWramMapImage

構文

#include <twl/mi.h>

MIWramImage MI_GetWramMapImage_A( void )
MIWramImage MI_GetWramMapImage_B( void )
MIWramImage MI_GetWramMapImage_C( void )

引数

なし。

返り値

指定のWRAM領域のイメージ出現設定です。
MI_WRAM_IMAGE_xxxKB で、xxx の部分は、
WRAM-A の場合 64, 128, 256 のいずれかで、
WRAM-B または WRAM-C の場合 32, 64, 128, 256 のいずれかです。

WRAM-A の場合

返り値 意味
MI_WRAM_IMAGE_64KB 64KB単位で WRAM-A のスロット0 のイメージを繰り返す
MI_WRAM_IMAGE_128KB 128KB単位で WRAM-A のスロット0〜1 のイメージを繰り返す
MI_WRAM_IMAGE_256KB 256KB単位で WRAM-A のスロット0〜3 のイメージを繰り返す



WRAM-B または WRAM-C の場合

返り値 意味
MI_WRAM_IMAGE_32KB 32KB単位で WRAM-B(WRAM-C) のスロット0 のイメージを繰り返す
MI_WRAM_IMAGE_64KB 64KB単位で WRAM-B(WRAM-C) のスロット0〜1 のイメージを繰り返す
MI_WRAM_IMAGE_128KB 128KB単位で WRAM-B(WRAM-C) のスロット0〜3 のイメージを繰り返す
MI_WRAM_IMAGE_256KB 256KB単位で WRAM-B(WRAM-C) のスロット0〜7 のイメージを繰り返す

説明

指定のWRAM領域のイメージ出現設定を取得します。

MI_GetWramMapImage_A() は WRAM-A について情報を取得します。
MI_GetWramMapImage_B() は WRAM-B について情報を取得します。
MI_GetWramMapImage_C() は WRAM-C について情報を取得します。

内部動作

IOレジスタのMBK6〜MBK8 にアクセスしています。
