MI_GetWramMapEnd*

構文

#include <twl/mi.h>

u32 MI_GetWramMapEnd_A( void )
u32 MI_GetWramMapEnd_B( void )
u32 MI_GetWramMapEnd_C( void )

引数

なし。

返り値

WRAM がマッピングされている領域の上位アドレスです。

説明

指定のWRAM領域がマッピングされている領域の上位アドレスを取得します。

MI_GetWramMapEnd_A() は WRAM-A について情報を取得します。
MI_GetWramMapEnd_B() は WRAM-B について情報を取得します。
MI_GetWramMapEnd_C() は WRAM-C について情報を取得します。

下位アドレスを取得する関数は、MI_GetWramMapStart*() です。

内部動作

IOレジスタのMBK6〜MBK8 にアクセスしています。
