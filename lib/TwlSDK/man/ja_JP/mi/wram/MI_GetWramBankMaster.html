MI_GetWramBankMaster*

構文

#include <twl/mi.h>

MIWramProc MI_GetWramBankMaster( MIWramPos wram, int num )
MIWramProc MI_GetWramBankMaster_A( int num )
MIWramProc MI_GetWramBankMaster_B( int num )
MIWramProc MI_GetWramBankMaster_C( int num )

引数

wram マスタープロセッサ情報を取得する対象の WRAM
num マスタープロセッサ情報を取得する対象 WRAM のスロット番号

返り値

指定のWRAM領域のマスタープロセッサです。
MI_WRAM_ARM9、MI_WRAM_ARM7、MI_WRAM_DSP のいずれかです。

説明

指定のWRAM領域のマスタープロセッサを取得します。

wram はマスタープロセッサ情報を取得する対象となる WRAM で、MIWramPos 型です。

num は対象の WRAM のスロット番号で、WRAM-A は 0〜3、WRAM-B と WRAM-C は 0〜7 となります。

MI_GetWramBankMaster_A() は MI_GetWramBankMaster( MI_WRAM_A, … ) の define です。
MI_GetWramBankMaster_B() は MI_GetWramBankMaster( MI_WRAM_B, … ) の define です。
MI_GetWramBankMaster_C() は MI_GetWramBankMaster( MI_WRAM_C, … ) の define です。

内部動作

IOレジスタのMBK1〜MBK5 にアクセスしています。
