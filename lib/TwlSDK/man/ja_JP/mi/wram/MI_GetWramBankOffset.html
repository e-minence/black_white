MI_GetWramBankOffset*

構文

#include <twl/mi.h>

MIWramOffset MI_GetWramBankOffset( MIWramPos wram, int num )
MIWramOffset MI_GetWramBankOffset_A( int num )
MIWramOffset MI_GetWramBankOffset_B( int num )
MIWramOffset MI_GetWramBankOffset_C( int num )

引数

wram オフセット情報を取得する対象の WRAM
num オフセット情報を取得する対象 WRAM のスロット番号

返り値

指定のWRAM領域のオフセット情報です。
MI_WRAM_OFFSET_xxxKB が返ります。 ただし、xxx の部分には、
WRAM-A の場合 0, 64, 128, 192 のいずれかが入ります。
WRAM-B または WRAM-C の場合 0, 32, 64, 96, 128, 160, 192, 224 のいずれかが入ります。

説明

指定のWRAM領域が割り当てられているオフセット値を取得します。

wram は情報を取得する対象となる WRAM で、MIWramPos 型です。

num は対象の WRAM のスロット番号で、WRAM-A は 0〜3、WRAM-B と WRAM-C は 0〜7 となります。

内部動作

IOレジスタのMBK1〜MBK5 にアクセスしています。
