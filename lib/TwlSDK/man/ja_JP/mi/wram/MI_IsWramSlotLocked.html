MI_IsWramSlotLocked*

構文

#include <twl/mi.h>

BOOL MI_IsWramSlotLocked_A( int num )
BOOL MI_IsWramSlotLocked_B( int num )
BOOL MI_IsWramSlotLocked_C( int num )

引数

num 情報を取得する、対象 WRAM のバンク番号

返り値

TRUE ならばロックされていることを示します。
FALSE ならばロックされていないことを示します。

説明

指定の WRAM スロットが、WRAM設定を変更できないようにロックされているかどうかを取得します。

num は指定の WRAM のうち、ロックされているかどうかを調べるスロットの番号です。

なお、WRAM設定のロックはアプリケーションが起動されるときにすでに設定がなされて、以降変更することが出来ません。起動後はこの関数で状態を取得することが出来るのみです。

内部動作

IOレジスタの MBK9 にアクセスしています。
