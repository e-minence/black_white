MI_IsWramSlotUsed*

構文

#include <twl/mi.h>

BOOL MI_IsWramSlotUsed( MIWramPos wram, int num )

(define)
BOOL MI_IsWramSlotUsed_A( int num )
BOOL MI_IsWramSlotUsed_B( int num )
BOOL MI_IsWramSlotUsed_C( int num )

引数

wram 情報を取得するWRAM
num 情報を取得する、対象 WRAM のslot番号

返り値

指定のWRAMスロットがどこかのプロセッサに確保されて使用されているかを表し、
TRUE ならば使用されています。FALSE ならばどこからも確保されていません。

説明

指定の WRAM スロットが、使用されているかを取得します。

num は指定の WRAM のうち、使用されているかどうかを調べるスロットの番号です。

指定の WRAM の情報を一気に調べる MI_GetUsedWramSlot*() も用意されています。

MI_IsWramSlotUsed_A() は MI_IsWramSlotUsed( MI_WRAM_A, ... ) の define です。
MI_IsWramSlotUsed_B() は MI_IsWramSlotUsed( MI_WRAM_B, ... ) の define です。
MI_IsWramSlotUsed_C() は MI_IsWramSlotUsed( MI_WRAM_C, ... ) の define です。
