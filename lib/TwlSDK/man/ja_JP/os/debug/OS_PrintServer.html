OS_PrintServer

構文

#include <nitro/os.h>

void OS_PrintServer( void );

引数

なし。

返り値

なし。

説明

ARM7 からのデバッグメッセージを表示するための関数です。

これは SDKライブラリのデバッグ用の関数ですので、通常のアプリケーションで使用する必要はありません。主にライブラリやミドルウェア開発者が使用することを想定しています。

この関数は、ARM7 が書き込んだ出力用のバッファの内容を書き出す関数で、定期的に(毎フレームに一回程度) 呼び出すようにしてください。詳細は OS_InitPrintServer() を参照してください。

OS_InitPrintServer() が呼び出されていない状況で OS_PrintServer() を呼び出しても何も行いません。

FINALROM ビルドでは、OS_InitPrintServer() および OS_PrintServer() は何もしないコードに置き換えられます。
