OS_GetAgreedEULAVersion

構文

#include <twl/os.h>

u8 OS_GetAgreedEULAVersion(void);

引数

なし。

返り値

同意したEULAバージョンです。( 0 〜 255 の数字です。)

説明

この本体で同意した EULA のバージョンを取得します。
EULA 同意済みかどうかは、OS_IsAgreeEULA() で調べることが出来ます。

ユーザーが同意している EULA(End User License Agreement) のバージョンが最新であるかは Wi-Fi コネクション接続時にTWL-DWC ライブラリが自動的にチェックしますので、通常アプリケーションがこの関数を呼び出す必要はありません。

参照

OSOwnerInfoEx, OS_IsAgreeEULA()
