OS_IsParentalControledApp

構文

#include <twl/os.h>

BOOL OS_IsParentalControledApp(u8* appRatingInfo);

引数

appRatingInfo レーティング情報へのポインタ。
アプリケーション ROM ヘッダのオフセット0x2f0 位置に埋め込まれている 16バイトがレーティング情報です。

返り値

TRUE の場合、ペアレンタルコントロール制限を掛けて、パスワード入力を促すべきということを表します。

FALSE の場合、制限をかけずに起動してよいアプリケーションであることを表します。

説明

アプリケーション起動に、ペアレンタルコントロール制限を掛けるべきかどうかを判定します。

appRatingInfo はレーティング情報へのポインタで、アプリケーション ROM ヘッダ内のオフセット 0x2f0 位置に埋め込まれているのでこれを示してください。
