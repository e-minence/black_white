OS_IsForceDisableWireless

構文

#include <twl/os.h>

BOOL OS_IsForceDisableWireless(void);

引数

なし。

返り値

生産時に無線通信の使用を禁止した TWL であるかの情報を取得します。

無線通信の使用を禁止した、特定用途向けの TWL で動作している場合には TRUE を、通常の TWL で動作している場合には FALSE を返します。

TWL モードで動作していない場合、常に FALSE を返します。

説明

生産時に無線通信の使用を禁止した TWL であるかの情報を取得します。

この関数は特殊な TWL 本体であるかを識別するため関数であり、通常のアプリケーションでは使用する必要はありません。
TWL の本体設定で設定可能な無線通信機能の ON/OFF の状態を取得したい場合は OS_IsAvailableWireless を利用してください。

参照

OSOwnerInfoEx, OS_IsAvailableWireless
