OS_GetOwnerInfo[Ex]

構文

OS_GetOwnerInfo

#include <nitro/os.h>
void OS_GetOwnerInfo( OSOwnerInfo* info );


OS_GetOwnerInfoEx

#include <twl/os.h>
void OS_GetOwnerInfoEx( OSOwnerInfoEx* info );

引数

info オーナー情報を取得するバッファへのポインタを指定します。

返り値

なし。

説明

NITRO 本体 や TWL 本体に保存されているエンドユーザーが入力したオーナー情報を取得します。

NITRO モードでオーナー情報を取得したい場合には OS_GetOwnerInfo() を、 TWL モードでオーナー情報を取得したい場合には OS_GetOwnerInfoEx() を使用してください。
なお、ハイブリッド ROM を NITRO 本体で実行した場合でも OS_GetOwnerInfoEx() を使用してオーナー情報を取得することが可能です。

参照

OS_GetMacAddress, OS_GetOwnerRtcOffset
