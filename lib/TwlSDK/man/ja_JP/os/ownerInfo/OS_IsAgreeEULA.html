OS_IsAgreeEULA

構文

#include <twl/os.h>

BOOL OS_IsAgreeEULA(void);

引数

なし。

返り値

この本体で EULA に同意済みかどうかで、TRUE ならば EULA 同意済みであることを表します。

FALSE の場合は非同意であることを表します。

TWL 上の動作でないときは常に FALSE を返します。

説明

この本体で EULA 同意済みかどうかを取得します。

同意した EULA のバージョンを取得するには OS_GetAgreedEULAVersion() を呼び出してください。

参照

OSOwnerInfoEx, OS_GetAgreedEULAVersion()
