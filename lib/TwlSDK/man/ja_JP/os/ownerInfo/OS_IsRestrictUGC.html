OS_IsRestrictUGC

構文

#include <twl/os.h>

BOOL OS_IsRestrictUGC(void);

引数

なし。

返り値

本体設定のペアレンタルコントロール で設定した情報を参照し、ユーザー作成コンテンツの送受信に関する制限がかけられている場合は TRUE を返します。
ペアレンタルコントロールの設定をしていない、もしくはユーザー作成コンテンツの送受信に関する制限がかけられていない場合は FALSE を返します。

説明

本体設定のペアレンタルコントロール で設定した情報を参照し、ユーザー作成コンテンツの送受信に関する制限がかけられているかどうかの判定をします。
ゲーム内で作成したコンテンツの送受信に関するの交換を行う場合には、事前にこの関数でチェックし、制限がかけられている場合にはパスワードの入力無しにコンテンツの交換が行われないようにしてください。
