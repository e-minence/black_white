OS_IsRestrictPhotoExchange

構文

#include <twl/os.h>

BOOL OS_IsRestrictPhotoExchange(void);

引数

なし。

返り値

本体設定のペアレンタルコントロール で設定した情報を参照し、写真データの交換に関する制限がかけられている場合は TRUE を返します。
ペアレンタルコントロールの設定をしていない、もしくは写真データの交換に関する制限がかけられていない場合は FALSE を返します。

説明

本体設定のペアレンタルコントロール で設定した情報を参照し、システムメニューやニンテンドー DSi カメラ で撮影した写真データの交換に関する制限がかけられているかどうかの判定をします。
ゲーム内で写真データの交換を行う場合には、事前にこの関数でチェックし、制限がかけられている場合にはパスワードの入力無しにデータの交換が行われないようにしてください。
