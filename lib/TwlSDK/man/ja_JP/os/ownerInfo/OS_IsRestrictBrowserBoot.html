OS_IsRestrictBrowserBoot

構文

#include <twl/os.h>

BOOL OS_IsRestrictBrowserBoot(void);

引数

なし。

返り値

本体設定のペアレンタルコントロール で設定した情報を参照し、ニンテンドー DSi ブラウザーの起動制限がかけられている場合は TRUE を返します。
ペアレンタルコントロールの設定をしていない、もしくはニンテンドー DSi ブラウザーの起動制限がかけられていない場合は FALSE を返します。

説明

本体設定のペアレンタルコントロール で設定した情報を参照し、ニンテンドー DSi ブラウザーの起動制限がかけられているかどうかの判定をします。
