OS_DumpThreadList

構文

#include <nitro/os.h>
void OS_DumpThreadList( void );

引数

なし

返り値

なし。

説明

スレッドシステムを用いている時に、現在のスレッドテーブルの内容を OS_Printf() を用いて表示します。

この関数はデバッグ用のものです。


表示モードについて

この関数の表示のように、全ての情報を確実に表示することが速度よりも優先される場合は、一時的にでも表示をブロッキングモードに設定することをお奨めします。詳細は OS_SetPrintBlockingMode() を参照してください。

参照

OS_InitThread, OS_CreateThread, OS_WakeupThread, OS_WakeupThreadDirect
OS_SetPrintBlockingMode()
