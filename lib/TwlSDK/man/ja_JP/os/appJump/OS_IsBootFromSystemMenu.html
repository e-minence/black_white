OS_IsBootFromSystemMenu

構文

#include <nitro/os.h>

BOOL OS_IsBootFromSystemMenu( void );

引数

なし。

返り値

現在の状態がシステムメニューから直接起動した後であった場合 TRUE を返します。
現在の状態がOS_RebootSystem()による再起動後であった場合には FALSE を返します。

説明

現在実行中のプログラムがシステムメニューから直接起動されたかをチェックします。
IS-TWL-DEBUGGER ソフトウェアで tad ファイルを実行した場合、TwlNmenu.srl から起動した場合にもシステムメニューから直接起動されたと判定します。

OS_RebootSystem()
