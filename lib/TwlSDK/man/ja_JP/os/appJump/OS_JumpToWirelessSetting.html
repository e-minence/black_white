OS_JumpToWirelessSetting

構文

#include <nitro/os.h>

BOOL OS_JumpToWirelessSetting( void );

引数

なし。

返り値

本体設定が存在しない開発環境で実行した場合など、遷移するために必要な条件を満たしていなかった場合 FALSE を返します。
成功した場合にはこの関数から戻ることはありません。

説明

ハードウェアリセットを行い、本体設定の "無線通信" の項目に遷移します。

遷移する際に、OS_JumpToSystemMenu() や PM_ForceToResetHardware() と同様のハードウェアリセットが行われますので、保存が必要なデータなどはこの関数を実行する前に保存しておいてください。

この関数を割り込み禁止の状態で呼び出すことはできません。
また、この関数は終了処理中に呼び出すことは出来ませんのでPM_Append*ExitCallbackなどで登録する終了コールバックから実行されないよう注意してください。

参照

PM_ForceToResetHardware, OS_JumpToSystemMenu, OS_JumpToInternetSetting, OS_JumpToEULAViewer,
PM_Append*ExitCallback, PM_Prepend*ExitCallback, PM_Insert*ExitCallback
