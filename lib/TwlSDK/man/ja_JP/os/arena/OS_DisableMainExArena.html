OS_DisableMainExArena

構文

#include <nitro/os.h>
void OS_DisableMainExArena( void );

引数

なし。

返り値

なし。

説明

メインメモリ上の拡張アリーナの不使用を宣言します。

ただし、物理的に拡張メインメモリがない状況では、常にメインメモリ拡張アリーナは使用できません。実機で動作する場合もそれに当てはまります。

この関数を呼ぶ場合は、OS_InitArena() や、OS_Init() の前に呼んでください。( OS_InitArena() は OS_Init() から呼ばれています。)

以前のバージョンとの互換性のために OS ライブラリのコンパイル時に NITRO_MEMSIZE=4M と指定する方法もあります。

OS_InitArena, OS_EnableMainExArena
