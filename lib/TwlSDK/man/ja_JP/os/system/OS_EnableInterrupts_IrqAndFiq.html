OS_EnableInterrupts_IrqAndFiq

構文

#include <nitro/os.h>

OSIntrMode OS_EnableInterrupts_IrqAndFiq( void );

引数

なし。

返り値

この関数を呼ぶ前の状態を表す値を返します。

返り値の型は OSIntrMode ( u32 の typedef ) ですが、返る値は以下の OSIntrMode_Irq 列挙型の値と OSIntrMode_Fiq列挙型の値の論理和となります。

OSIntrMode_Irq 列挙型

値 以前の状態
OS_INTRMODE_IRQ_DISABLE IRQディセーブル(禁止)
OS_INTRMODE_IRQ_ENABLE IRQイネーブル(許可)


OSIntrMode_Fiq 列挙型

値 以前の状態
OS_INTRMODE_FIQ_DISABLE FIQディセーブル(禁止)
OS_INTRMODE_FIQ_ENABLE FIQイネーブル(許可)

説明

IRQ と FIQ の割り込みを許可します。

ここで許可するのは、すべての割り込みのマスターイネーブルです。IRQ割り込み要因のイネーブルを個別にセットしていても、この関数でIRQ が許可になっていないと割り込みは発生しません。

返り値は、この関数を呼ぶ以前の状態を表す値です。

内部動作

CPSRレジスタ(current program status register)の IRQとFIQ割り込みの制御ビットをリセット(リセットがイネーブル)にしています。
