OS_IsCodecTwlMode

構文

#include <nitro/os.h>

BOOL OS_IsCodecTwlMode( void );

引数

なし。

返り値

TRUE なら CODEC は TWL モードで動作していることを表します。

FALSE ならそれ以外のモード(すなわち NITRO モード)で動作していることを表します。

NITRO 専用 ROM ではこの関数は定数 FALSE に置き換わります。

説明

CODECが TWLモードで動作しているかどうかを調べます。

TWL/NITRO HYBRID ROM か、TWL 専用 ROM では動作モードや ROM ヘッダ等を調べて値を返しますが、NITRO 専用 のROM では関数自体が定数 FALSE に置き換わるようになっています。
