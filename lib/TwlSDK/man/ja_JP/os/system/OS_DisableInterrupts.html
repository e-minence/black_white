OS_DisableInterrupts

構文

#include <nitro/os.h>

OSIntrMode OS_DisableInterrupts( void );

引数

なし。

返り値

この関数を呼ぶ前の状態を表す値を返します。

返り値の型は OSIntrMode ( u32 の typedef ) ですが、返り得る値は以下の OSIntrMode_Irq 列挙型　の値のどちらかです。

値 以前の状態
OS_INTRMODE_IRQ_DISABLE IRQディセーブル(禁止)
OS_INTRMODE_IRQ_ENABLE IRQイネーブル(許可)

説明

IRQ 割り込みを禁止します。

ここで禁止するのは、すべての割り込みのマスターイネーブルです。IRQ割り込み要因のイネーブルを個別にセットしていても、この関数でIRQを禁止していると割り込みは発生しません。

返り値は、この関数を呼ぶ以前の状態を表す値です。

内部動作

CPSRレジスタ(current program status register)の IRQ割り込みの制御ビットをセット(セットがディセーブル)にしています。

OS_EnableInterrupts, OS_RestoreInterrupts
