OS_InitChina

構文

#include <nitro/os.h>
void OS_InitChina( const char** isbn, OSChinaIsbn param );

引数

isbn 健康上の注意画面に表示する ISBN 番号等の文字列を4つ指定します。 文字列のフォーマットは以下のようになっています。なお、数字及び '-' 以外の文字は全て ' ' (スペース) として扱われます。
0
 char[ 17 ] "***-*-******-**-*" ISBN 番号
1
 char[ 12 ] "***-****-***" 合同登記号
2
 char[ 4 ] "****" 新出音管(左側に表示する情報)
3
 char[ 3 ] "****" 新出音管(右側に表示する情報)
param ISBN 番号表示に関する動作を指定します。
OS_CHINA_ISBN_DISP ISBN 番号を表示します。
OS_CHINA_ISBN_NO_DISP ISBN 番号を表示しません。NAND アプリケーションのときに指定してください。
OS_CHINA_ISBN_CHECK_ROM 検閲用 ROM を作成するときに指定してください。ISBN 番号は必ず表示され、健康上の注意画面でボタン入力待ちになります。

返り値

なし。

説明

OS_Init()と同様にシステムを初期化した後、ハードウェアの起動制限を確認します。
中国向けアプリケーションが中国を仕向地としていない本体から起動された場合に、 アプリケーションの起動が制限され、この関数からは制御が戻されません。
中国向けアプリケーションが中国を仕向地とする本体から正しく起動された場合には、この関数内で健康上の注意を喚起すると共に ISBN 番号等の表示を義務付けられた情報を告知する画面表示が行われます。
中国向けアプリケーションを作成する為には、OS_Init()の代わりに当関数を使用するほか、 ROMイメージ内に中国語対応であることを示す付帯情報を定義する必要があります。 詳細は makerom の説明を参照して下さい。

システムの初期化については OS_Init()の説明を参照して下さい。
