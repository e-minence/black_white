OSBootInfo

定義

#include <nitro/os.h>
typedef struct OSBootInfo
{
    OSBootType   boot_type;
    u16          length;
    u16          rssi;
    u16          bssid[3];
    u16          ssidLength;
    u16          ssid[32];
    u16          capaInfo;
    struct
    {
        u16      basic;
        u16      support;
    }
    rateSet;
    u16          beaconPeriod;
    u16          dtimPeriod;
    u16          channel;
    u16          cfpPeriod;
    u16          cfpMaxDuration;
    u16          rsv1;
}
OSBootInfo;

要素

boot_type ブート種別を指す OSBootType 列挙体です。
length ライブラリ内部で使用されます。
内容は、マルチブート子機にとっての親機情報を表す MBParentBssDesc 構造体と一致します。
