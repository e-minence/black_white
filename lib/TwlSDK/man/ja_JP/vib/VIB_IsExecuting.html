VIB_IsExecuting

構文

#include <nitro/vib.h>
BOOL VIB_IsExecuting(void);

引数

なし。

返り値

VIB_StartPulse( )を呼んでパルス振動を開始した時点から VIB_StopPulse( )を呼んでパルス振動を停止させるまでの間 TRUE を返し、
それ以外の場合は FALSE を返します。

説明

パルス振動を行っているか否かを返します。

repeat_num 回振動後、振動は停止していても VIB_StopPulse( )が呼ばれない限りは TRUE を返します。
