TCL_IsSameImageType

定義

#include <twl/tcl.h>


BOOL TCL_IsSameImageType( const TCLPictureInfo* pPicInfo , const TCLMakerNote* pMakerNote );

引数

pPicInfo写真情報
pMakerNoteメーカーノート構造体

返り値

同じ画像タイプとみなせる場合 TRUE

説明

管理ファイル上での画像タイプと、画像をデコードして得られるメーカーノートから取得できる画像タイプ(メーカーノートを TCLMakerNote 型に変換して比較対象とします)が同じかどうか判別します。

比較のための処理は概要を参照してください。

実際には管理ファイル上では画像タイプが写真か合成写真か分からない状態、というのが存在するため、それらの場合も同じとみなします。
