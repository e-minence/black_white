TCLSearchObject

定義

#include <twl/tcl.h>


typedef struct
{
    int startIdx;
    u32 condition;
} TCLSearchObject;

説明

検索関連の関数に使用する構造体です。

TCL_InitSearchObject を使用して初期化してから使用します。

各メンバをアプリ側で操作することは禁止されています。

参照

TCL_InitSearchObject TCL_SearchNextPictureInfo TCL_SearchPictureInfoByIndex TCL_SearchNextPicturePath TCL_SearchPicturePathByIndex
