TCL_SearchPictureInfoByIndex

定義

#include <twl/tcl.h>


TCLResult TCL_SearchPictureInfoByIndex( const TCLAccessor* pAccessor ,
                                        const TCLPictureInfo** ppPicInfo ,
                                        const TCLSearchObject* pSearchObj ,
                                        int index );

引数

pAccessorアクセサ
ppPicInfo見つかった写真情報を格納するポインタ
pSearchObj検索オブジェクト
index条件に合致した中で実際に取得する写真情報のインデクス

返り値

処理結果

説明

検索オブジェクトで表される検索条件に全て合致する写真情報の中から index で表される順番の写真情報を取得します。

返り値が TCL_RESULT_SUCCESS 以外の場合は取得される写真情報の値は不定です。

検索処理中に TCL_EncodeAndWritePicture や TCL_SortTable 、TCL_DeletePicture など、管理される写真の枚数の増減や順番変更が発生するような処理を行った場合の挙動は不定です。

アクセサは TCL_LoadTable や TCL_CreateTable などで初期化されている必要があります。
