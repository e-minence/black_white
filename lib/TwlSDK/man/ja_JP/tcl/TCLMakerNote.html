TCLMakerNote

定義

#include <twl/tcl.h>


typedef struct
{
    union
    {
        u32 bitField;
        struct
        {
            u32 imageType :  2;
            u32 reserved0 : 30;
        };
    };
    u32 reserved1;
} TCLMakerNote;

説明

DSiカメラで読み込める形式のjpeg画像を作成するため Exif 情報のメーカーノートタグに埋め込んでいただく構造体です。

エンコード時に使用する TCL_EncodeAndWritePicture(Ex) は、内部でメーカーノートへ埋め込みますので、直接この構造体を意識する必要はありません。

デコードしたjpeg画像の正当性を評価するために必要になる場合があります。詳しくは TCL_IsSameImageType を参照してください。

各メンバをアプリ側で操作することは禁止されています。

参照

TCL_EncodeAndWritePicture(Ex) TCL_IsSameImageType
