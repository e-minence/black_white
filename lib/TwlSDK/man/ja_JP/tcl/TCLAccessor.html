TCLAccessor

定義

#include <twl/tcl.h>


typedef struct
{
    void* tableBuffer;
    u32   tableBufferSize;
    void* workBuffer;
    u32   workBufferSize;
} TCLAccessor;

説明

TCLの各関数にアクセスするときに使用するアクセサです。

TCL_LoadTable や TCL_CreateTable など、管理ファイルの読み込み・生成を行う関数を読み出した際に初期化されます。

各メンバをアプリ側で操作することは禁止されています。

その後の各APIへのアクセスはこのアクセサを使用してください。

参照

TCL_LoadTable TCL_CreateTable
