TCL_PrintPicturePath

定義

#include <twl/tcl.h>


int TCL_PrintPicturePath( char* path ,
                          size_t len ,
                          const TCLPictureInfo* pPicInfo );

引数

pathパス格納バッファ
lenパス長
pPicInfo写真情報

返り値

正しくパスを取得できた場合の文字列長(STD_TSNPrintf の返り値)

説明

検索関数で取得した写真情報からパスを取得します。
