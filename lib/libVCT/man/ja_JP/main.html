はじめに

VoiceChatライブラリを使うことで、ニンテンドーDS/DSiのワイヤレス通信（インフラストラクチャーモード）を利用してインターネットに接続し、他のニンテンドーDS/DSiのユーザーと音声通話を行うことが出来ます。

VoiceChatライブラリは、下記のライブラリが必要です。

  • TWL-SDKライブラリ 5.3 + patch1(2009/08/24版)
  • TWL-Systemライブラリ 2.2(2009/08/05版)
  • TWL-DWCライブラリ 5.3

また、下記の環境で動作テストしています。

  • CodeWarrior for DSi ver.1.1(2008/12/09版) + patch3(2009/08/19版)
  • TWL-SDKライブラリ 5.3 + patch1(2009/08/24版)
  • TWL-Systemライブラリ 2.2(2009/08/05版)
  • TWL-DWCライブラリ 5.3

このパッケージには、ソースの含まれないバイナリが含まれますので、ご注意ください。

使用方法

TWLLIBVCT_ROOT 環境変数に、ライブラリへのパスを設定して下さい。

注意事項

VoiceChatライブラリを使用する場合は、TWL-DWCを利用したマッチメイクが必須となります。

API の使用

プログラムのソースコードには、以下のヘッダファイルをインクルードして下さい。


	#inculude <vct.h>

アプリケーション・ライブラリを作成する Makefile には、以下の共通ビルドツールをインクルードして下さい。


	include $(TWLLIBVCT_ROOT)/build/buildtools/commondefs
	include $(TWLLIBVCT_ROOT)/build/buildtools/modulerules

このビルドツールをインクルードするとTWL-DWCのビルドツールもインクルードされます。
