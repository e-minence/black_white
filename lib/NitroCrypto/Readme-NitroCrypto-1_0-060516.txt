1. はじめに
==============================================================================
    NITRO-SDK Cryptoライブラリ は、ニンテンドーDS用ゲームソフトの開発に使用で
    きる、セキュリティライブラリの総称です。ライブラリ・パッケージには下記に示
    すものが含まれています。
    
    ・暗号化ライブラリ
    ・電子署名ライブラリ
    ・関数リファレンス及びサンプルデモ
    
    詳細に関しては、付属のオーバービュー及び関数リファレンスをご参照下さい。
    
    本パッケージは以下の開発環境で動作確認が行われています。
    
    ・NITRO-SDK 3.0 (060125) ＋ plus2パッチ (060307) 
      CodeWarrior for NINTENDO DS version 1.2 SP3 patch2
    
    ・NITRO-SDK 3.1 (060511)
      CodeWarrior for NINTENDO DS version 2.0 patch1

2. インストール
==============================================================================
    パッケージを任意のディレクトリに解凍した後、環境変数「NITROCRYPTO_ROOT」に
    NitroCryptoへのパスを指定して下さい。
    次に、以下のいずれかの方法で、libcrypto.a をリンク可能な状態にして下さい。
      
    ・NitroCrypto のルートディレクトリで make attach コマンドを実行する。
      
      上記により $NitroSDK/build/buildtools 以下に commondefs.add-ins.crypto
      が追加されます。NITRO-SDKが更新されるたび、コマンドを再実行しなければら
      無いことに注意して下さい。
      このファイルを削除したい場合は、make detach コマンドを実行して下さい。
      
    ・Makefile を使用している場合、以下の行を Makefile に追加する。
      
      LLIBARIES += libcrypto$(NITRO_LIBSUFFIX).a
        
      IDE を使用している場合、libcrypto.a または libcrypto.thumb.a を手動でプ
      ロジェクトに追加して下さい。


3. 注意事項
==============================================================================
    NITRO-SDK Cryptoライブラリは、「外国為替及び外国貿易法」「輸出貿易管理令」
    「外国為替令」などで定められた暗号装置としての機能を搭載しているため、日本
    から輸出する場合は、事前に経済産業省へ輸出許可申請を行い許可を得ておく必要
    があります。また、海外から他国へ輸出する場合には各国の輸出規制関連法を遵守
    しなければなりませんので、ご使用の際はご注意下さい。
